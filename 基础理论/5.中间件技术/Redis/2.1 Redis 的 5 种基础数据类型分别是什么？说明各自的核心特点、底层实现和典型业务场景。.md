**标准答案**：

表格

|数据类型|核心特点|底层实现|典型业务场景|
|:--|:--|:--|:--|
|String（字符串）|二进制安全，单 value 最大支持 512MB，支持位操作|int（整数）、embstr（短字符串）、raw（长字符串）|通用缓存、计数器、分布式锁、Session 存储|
|Hash（哈希）|字段 - 值映射结构，适合存储对象，支持单字段精准读写|列表 pack（listPack，7.0+）/ 压缩列表（zipList）、哈希表|商品 / 用户信息存储、购物车|
|List（列表）|有序可重复，支持双向头尾高效操作，可模拟队列 / 栈|快速列表（quickList）、列表 pack|消息队列、Timeline 流、排行榜分页|
|Set（集合）|无序不可重复，原生支持交、并、差集操作|整数集合（intSet）、哈希表|点赞 / 收藏、好友推荐、共同关注|
|Sorted Set（ZSet 有序集合）|不可重复，按 score 排序，支持范围排名查询|跳表（skipList）+ 哈希表、列表 pack|实时排行榜、延时队列、范围筛选|
