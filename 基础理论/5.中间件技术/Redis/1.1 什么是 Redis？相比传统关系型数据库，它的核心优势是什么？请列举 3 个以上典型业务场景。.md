**标准答案**：

Redis 是一款基于**内存**的开源键值型 NoSQL 数据库，支持数据持久化，具备高性能、高可用、丰富的数据结构特性。

- 核心优势：
    
    1. 极致性能：纯内存操作，核心命令执行采用单线程模型（避免多线程锁开销），单机 QPS 可达 10 万 +；
    2. 丰富的数据结构：支持 String、Hash、List、Set、ZSet 等多种数据类型，适配复杂业务场景；
    3. 原生原子性：所有单命令均为原子性，支持 Lua 脚本实现批量复杂原子操作；
    4. 高可用与分布式：原生支持主从复制、哨兵、Cluster 集群，满足大规模分布式场景需求。
    
- 典型场景：分布式缓存、分布式锁、计数器、排行榜、用户会话存储、轻量消息队列、接口限流。
