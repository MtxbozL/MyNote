#### 参考答案

Nginx 是一款**高性能、轻量级、异步非阻塞**的 HTTP 和反向代理服务器，同时也支持邮件代理、TCP/UDP 代理，核心优势是高并发、低内存占用、高稳定性、配置简单、扩展性强。

- 核心特性：事件驱动的异步非阻塞模型、多进程架构、热部署、丰富的第三方模块、跨平台兼容。
- 核心应用场景：
    
    1. 反向代理（最核心）：隐藏后端服务，统一入口，做请求转发
    2. 负载均衡：将请求分发至多台后端服务器，提升系统并发和可用性
    3. 动静分离：直接处理静态资源（图片、JS、CSS、HTML），转发动态请求至后端应用，降低服务端压力
    4. 静态 Web 服务器：托管静态网站 / 资源，性能远超 Apache 等传统服务器
    5. HTTPS/SSL 终端：统一处理 SSL 证书加解密，简化后端服务配置
    6. 安全防护与限流：实现 IP 黑白名单、接口限流、防 CC 攻击、WAF 基础防护
    7. 缓存加速：对静态资源 / 接口响应做本地缓存，减少后端请求
