#### 以  `vector<vector<string>>` 为例。

`vector<vector<string>>` 是一个 **「嵌套的动态数组」**：

- 最外层是一个 `vector`，里面装的不是普通元素，而是**一个个「小 `vector<string>`」**；
- 每个「小 `vector<string>`」又是一个动态数组，里面装的是**字符串（`string`）**。

简单类比：它就像一个「**装文件夹的大文件夹**」—— 大文件夹（外层 `vector`）里放着多个小文件夹（内层 `vector<string>`），每个小文件夹里装着若干份文档（`string` 字符串）。

---

## 二、逐层拆解（从基础到嵌套，新手友好）

要理解 `vector<vector<string>>`，我们需要从最基础的 `vector` 开始，逐层往上嵌套。

### 第 1 层：基础 `std::vector`（动态数组）

`std::vector` 是 C++ 标准库的**动态数组**，可以自动扩容，存储任意类型的元素。

```cpp
#include <vector>
#include <string>

// 示例：一个存储整数的 vector
std::vector<int> intVec;
intVec.push_back(10);
intVec.push_back(20);
// intVec 现在是 [10, 20]
```

---

### 第 2 层：`std::vector<std::string>`（字符串数组）

把 `vector` 的元素类型指定为 `std::string`，就变成了 **「存储字符串的动态数组」**。

```cpp
#include <vector>
#include <string>

// 示例：一个存储字符串的 vector
std::vector<std::string> strVec;
strVec.push_back("eat");
strVec.push_back("tea");
strVec.push_back("ate");
// strVec 现在是 ["eat", "tea", "ate"]（一组字母异位词）
```

---

### 第 3 层：`std::vector<std::vector<std::string>>`（嵌套数组，即你代码中的类型）

再把外层 `vector` 的元素类型指定为 `std::vector<std::string>`，就变成了 **「存储『字符串数组』的数组」**—— 也就是你代码中的 `vector<vector<string>>`。

```cpp
#include <vector>
#include <string>

// 你的代码定义：
std::vector<std::vector<std::string>> result;
```

## 三、结合你的代码场景：为什么需要这个类型？

回到你之前做的 **LeetCode 49. 字母异位词分组** 题目，题目要求的输出是：

> 输入：`["eat","tea","tan","ate","nat","bat"]`
> 
> 输出：`[["bat"],["nat","tan"],["ate","eat","tea"]]`

这个输出的结构正好对应 `vector<vector<string>>`：

- 最外层 `vector`：装所有的「分组」（对应 `[["bat"],["nat","tan"],["ate","eat","tea"]]` 整个大括号）；
- 内层每个 `vector<string>`：装**同一组的字母异位词**（对应 `["bat"]`、`["nat","tan"]`、`["ate","eat","tea"]` 这三个小分组）。

## 四、具体代码示例（直观展示怎么用）

### 1. 往 `vector<vector<string>>` 中插入数据

```cpp
#include <vector>
#include <string>
#include <iostream>

int main() {
    // 1. 定义嵌套 vector
    std::vector<std::vector<std::string>> result;

    // 2. 创建第一个小分组（字母异位词组1）
    std::vector<std::string> group1;
    group1.push_back("bat");
    // 把 group1 插入到外层 result 的尾部
    result.push_back(group1);

    // 3. 创建第二个小分组（字母异位词组2）
    std::vector<std::string> group2;
    group2.push_back("nat");
    group2.push_back("tan");
    // 把 group2 插入到外层 result 的尾部
    result.push_back(group2);

    // 4. 创建第三个小分组（字母异位词组3）
    std::vector<std::string> group3;
    group3.push_back("ate");
    group3.push_back("eat");
    group3.push_back("tea");
    // 把 group3 插入到外层 result 的尾部
    result.push_back(group3);

    // 此时 result 的结构就是：
    // [
    //   ["bat"],
    //   ["nat", "tan"],
    //   ["ate", "eat", "tea"]
    // ]

    return 0;
}
```

---

### 2. 遍历 `vector<vector<string>>`（读取数据）

需要用**两层循环**：外层循环遍历「每个小分组」，内层循环遍历「分组里的每个字符串」。

```cpp
#include <vector>
#include <string>
#include <iostream>

int main() {
    // 假设 result 已经按上面的代码插入了数据
    std::vector<std::vector<std::string>> result = {
        {"bat"},
        {"nat", "tan"},
        {"ate", "eat", "tea"}
    };

    // 两层范围 for 循环遍历（推荐，C++11+）
    // 外层循环：遍历每个小分组（group 是 vector<string>）
    for (const auto& group : result) {
        std::cout << "分组：";
        // 内层循环：遍历分组里的每个字符串（s 是 string）
        for (const auto& s : group) {
            std::cout << s << " ";
        }
        std::cout << std::endl;
    }

    // 输出：
    // 分组：bat 
    // 分组：nat tan 
    // 分组：ate eat tea 

    return 0;
}
```

## 五、回到你的 `groupAnagrams` 代码：完整上下文

结合你之前的代码，`vector<vector<string>>` 是**函数的返回值类型**，用来存储最终的分组结果：

```cpp
#include <vector>
#include <string>
#include <unordered_map>
#include <algorithm>

using namespace std;

class Solution {
public:
    // 函数返回值是 vector<vector<string>>：返回所有分组
    vector<vector<string>> groupAnagrams(vector<string>& strs) {
        unordered_map<string, vector<string>> hashMap;
        
        // 第一步：遍历字符串，用哈希表分组
        for (const string& s : strs) {
            string key = s;
            sort(key.begin(), key.end());
            hashMap[key].push_back(s); // 把当前字符串加入对应分组
        }
        
        // 第二步：把哈希表中的分组收集到 result 中
        vector<vector<string>> result; // 定义嵌套 vector 存储最终结果
        for (auto& pair : hashMap) {
            // pair.second 就是哈希表中每个 key 对应的小分组（vector<string>）
            result.push_back(pair.second); // 把小分组插入到外层 result
        }
        
        return result; // 返回嵌套 vector，匹配题目要求的输出格式
    }
};
```

## 六、总结

1. **核心结构**：`vector<vector<string>>` 是「嵌套的动态数组」，外层装「分组」，内层装「每个分组里的字符串」；
2. **使用场景**：当需要存储 “一组组数据” 时（如字母异位词分组、二维数组、多组测试结果等），就需要嵌套 vector；
3. **插入数据**：先创建内层 `vector<string>`，再用 `push_back()` 把它插入到外层 `vector<vector<string>>`；
4. **遍历数据**：需要两层循环，外层遍历分组，内层遍历分组里的字符串。
