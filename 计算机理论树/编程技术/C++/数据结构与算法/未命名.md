## 一、接雨水的核心原理（所有解法的基础，必须先理解）

对于数组中**除首尾外的每一个位置 i**，它能接住的雨水量，只取决于两个值：

- `max_left[i]`：i 位置**左侧所有柱子的最大高度**
- `max_right[i]`：i 位置**右侧所有柱子的最大高度**

公式：

`当前位置i的雨水量 = max( 0, min(max_left[i], max_right[i]) - height[i] )`

总雨水量 = 所有位置的雨水量之和

> 通俗理解：一个位置能装多少水，看左右两边最高的 “挡板”，矮的那个挡板决定了水位上限，减去当前柱子的高度，就是能装的水量。如果结果是负数，说明装不了水，记为 0。

---

## 三、正确的解法（从易到难，面试优先掌握）

### 解法 1：暴力法（入门理解，吃透核心原理）

#### 思路

遍历每一个位置 i，对每个 i 分别遍历左右两侧，找到左侧最大高度和右侧最大高度，按公式计算雨水量，累加得到结果。

#### 完整 C++ 代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm> // max/min函数所需头文件

using namespace std;

class Solution {
public:
    int trap(vector<int>& height) {
        int n = height.size();
        if (n < 3) return 0; // 小于3个柱子，无法形成凹槽，接不到水
        int totalWater = 0;

        // 首尾柱子接不到水，遍历1 ~ n-2的位置
        for (int i = 1; i < n - 1; ++i) {
            // 找i左侧的最大高度
            int maxLeft = 0;
            for (int j = 0; j < i; ++j) {
                maxLeft = max(maxLeft, height[j]);
            }
            // 找i右侧的最大高度
            int maxRight = 0;
            for (int j = i + 1; j < n; ++j) {
                maxRight = max(maxRight, height[j]);
            }
            // 计算当前位置的雨水量，累加
            int currentWater = min(maxLeft, maxRight) - height[i];
            if (currentWater > 0) {
                totalWater += currentWater;
            }
        }
        return totalWater;
    }
};

// 测试代码
int main() {
    Solution solution;
    // 示例1
    vector<int> height1 = {0,1,0,2,1,0,1,3,2,1,2,1};
    cout << "示例1结果：" << solution.trap(height1) << endl; // 预期输出6
    // 示例2
    vector<int> height2 = {4,2,0,3,2,5};
    cout << "示例2结果：" << solution.trap(height2) << endl; // 预期输出9
    return 0;
}
```

#### 复杂度分析

- 时间复杂度：`O(n²)`，n 为数组长度，每个位置都要遍历左右两侧，最坏情况需要两层循环
- 空间复杂度：`O(1)`，仅使用常数级额外空间
- 缺点：大数据量会超时，仅用于理解核心原理，面试不推荐写这个

### 解法 2：动态规划法（优化暴力，面试常写，无超时风险）

#### 思路

暴力法中，每个位置都要重复计算左右最大高度，我们可以提前用两个数组`maxLeft`和`maxRight`，一次性预处理出所有位置的左右最大高度，避免重复计算。

1. 从左到右遍历，计算`maxLeft[i]`：`maxLeft[i] = max(maxLeft[i-1], height[i-1])`（i 位置左侧的最大高度，等于 i-1 位置的左侧最大高度 和 i-1 位置高度的较大值）
2. 从右到左遍历，计算`maxRight[i]`：`maxRight[i] = max(maxRight[i+1], height[i+1])`
3. 最后遍历每个位置，按公式计算雨水量累加即可

#### 完整 C++ 代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

class Solution {
public:
    int trap(vector<int>& height) {
        int n = height.size();
        if (n < 3) return 0;
        int totalWater = 0;

        // 预处理：maxLeft数组，maxLeft[i]表示i左侧的最大高度
        vector<int> maxLeft(n, 0);
        for (int i = 1; i < n; ++i) {
            maxLeft[i] = max(maxLeft[i-1], height[i-1]);
        }

        // 预处理：maxRight数组，maxRight[i]表示i右侧的最大高度
        vector<int> maxRight(n, 0);
        for (int i = n-2; i >= 0; --i) {
            maxRight[i] = max(maxRight[i+1], height[i+1]);
        }

        // 遍历计算每个位置的雨水量
        for (int i = 1; i < n-1; ++i) {
            int currentWater = min(maxLeft[i], maxRight[i]) - height[i];
            if (currentWater > 0) {
                totalWater += currentWater;
            }
        }
        return totalWater;
    }
};

// 测试代码和暴力法完全一致
int main() {
    Solution solution;
    vector<int> height1 = {0,1,0,2,1,0,1,3,2,1,2,1};
    cout << "示例1结果：" << solution.trap(height1) << endl;
    vector<int> height2 = {4,2,0,3,2,5};
    cout << "示例2结果：" << solution.trap(height2) << endl;
    return 0;
}
```

#### 复杂度分析

- 时间复杂度：`O(n)`，仅需 3 次遍历数组，无嵌套循环
- 空间复杂度：`O(n)`，需要两个数组存储左右最大高度
- 优点：逻辑清晰、无超时风险，面试中写这个不会出错，是最稳妥的写法

### 解法 3：双指针法（最优解，面试必掌握）

#### 思路

动态规划法用了两个数组存储左右最大高度，我们可以用两个指针 + 两个变量，把空间复杂度优化到`O(1)`。

核心结论：

- 用左指针`left`从左往右走，右指针`right`从右往左走
- 用`leftMax`记录左指针左侧的最大高度，`rightMax`记录右指针右侧的最大高度
- 如果`height[left] < height[right]`：说明左指针位置的水位上限由`leftMax`决定（因为右侧有更高的挡板），直接计算左指针位置的雨水量，左指针右移
- 反之：说明右指针位置的水位上限由`rightMax`决定，计算右指针位置的雨水量，右指针左移

#### 完整 C++ 代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

class Solution {
public:
    int trap(vector<int>& height) {
        int n = height.size();
        if (n < 3) return 0;

        int left = 0; // 左指针，从最左侧开始
        int right = n - 1; // 右指针，从最右侧开始
        int leftMax = 0; // 左指针左侧的最大高度
        int rightMax = 0; // 右指针右侧的最大高度
        int totalWater = 0;

        while (left < right) {
            if (height[left] < height[right]) {
                // 左指针位置的水位由leftMax决定
                if (height[left] >= leftMax) {
                    leftMax = height[left]; // 更新左侧最大高度
                } else {
                    // 计算当前位置的雨水量，累加
                    totalWater += leftMax - height[left];
                }
                left++; // 左指针右移
            } else {
                // 右指针位置的水位由rightMax决定
                if (height[right] >= rightMax) {
                    rightMax = height[right]; // 更新右侧最大高度
                } else {
                    // 计算当前位置的雨水量，累加
                    totalWater += rightMax - height[right];
                }
                right--; // 右指针左移
            }
        }
        return totalWater;
    }
};

// 测试代码
int main() {
    Solution solution;
    vector<int> height1 = {0,1,0,2,1,0,1,3,2,1,2,1};
    cout << "示例1结果：" << solution.trap(height1) << endl;
    vector<int> height2 = {4,2,0,3,2,5};
    cout << "示例2结果：" << solution.trap(height2) << endl;
    return 0;
}
```

#### 复杂度分析

- 时间复杂度：`O(n)`，仅需一次遍历数组，左右指针相遇即结束
- 空间复杂度：`O(1)`，仅使用 4 个变量，空间最优
- 优点：时间和空间都是最优，是面试中最加分的写法，也是这道题的最优解

### 解法 4：单调栈法（进阶，按行计算）

#### 思路

用单调递减栈存储柱子的下标，遍历数组时，当遇到比栈顶元素高的柱子，说明形成了凹槽，可以计算凹槽的雨水量。这种方法是**按行计算**，和前面按列计算的思路不同，适合拓展理解。

#### 完整 C++ 代码

```cpp
#include <iostream>
#include <vector>
#include <stack>
#include <algorithm>

using namespace std;

class Solution {
public:
    int trap(vector<int>& height) {
        int n = height.size();
        if (n < 3) return 0;
        int totalWater = 0;
        stack<int> st; // 单调递减栈，存储柱子的下标

        for (int i = 0; i < n; ++i) {
            // 遇到比栈顶高的柱子，形成凹槽，计算雨水量
            while (!st.empty() && height[i] > height[st.top()]) {
                int mid = st.top(); // 凹槽底部的下标
                st.pop();
                if (st.empty()) break; // 没有左挡板，接不到水

                int left = st.top(); // 左挡板下标
                int width = i - left - 1; // 凹槽的宽度
                // 水位高度 = 左右挡板的较小值 - 凹槽底部的高度
                int waterHeight = min(height[left], height[i]) - height[mid];
                totalWater += width * waterHeight;
            }
            st.push(i); // 当前柱子入栈，维持单调递减
        }
        return totalWater;
    }
};

// 测试代码
int main() {
    Solution solution;
    vector<int> height1 = {0,1,0,2,1,0,1,3,2,1,2,1};
    cout << "示例1结果：" << solution.trap(height1) << endl;
    vector<int> height2 = {4,2,0,3,2,5};
    cout << "示例2结果：" << solution.trap(height2) << endl;
    return 0;
}
```

#### 复杂度分析

- 时间复杂度：`O(n)`，每个元素入栈和出栈各一次
- 空间复杂度：`O(n)`，最坏情况栈需要存储所有元素
