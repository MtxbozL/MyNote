## 一、题目核心要求拆解

1. 核心定义：**字母异位词**，指两个字符串的字母组成、每个字母的出现次数完全一致，仅字符排列顺序不同。
2. 输入：字符串数组 `strs`
3. 输出：二维字符串数组，将所有互为字母异位词的字符串分到同一个子数组中，返回顺序、子数组内的顺序均无要求
4. 边界场景：空字符串、单个字符、重复字符串均需正常处理

---

## 二、解法 1：排序哈希法（最直观、面试首选）

### 核心思路

互为字母异位词的字符串，对其内部字符排序后，得到的字符串**完全相同**。

例如：`eat`、`tea`、`ate` 排序后均为 `aet`，因此可以将「排序后的字符串」作为哈希表的唯一 key，将原字符串按 key 分组存入哈希表，最终提取所有分组即可。

### 详细执行流程

1. 初始化哈希表：`unordered_map<string, vector<string>>`，其中 key 为排序后的字符串，value 为对应字母异位词的分组列表。
2. 遍历输入数组中的每一个字符串：
    
    a. 拷贝当前字符串，对拷贝后的字符串执行字符升序排序，得到分组 key。
    
    b. 将原字符串追加到哈希表中该 key 对应的 vector 列表末尾。
3. 遍历完成后，将哈希表中所有的 value（各个分组列表）提取出来，组装成最终的二维数组返回。

### 完整 C++ 代码

运行

```cpp
#include <iostream>
#include <vector>
#include <unordered_map>
#include <algorithm> // 排序所需头文件

using namespace std;

class Solution {
public:
    vector<vector<string>> groupAnagrams(vector<string>& strs) {
        // 哈希表：key=排序后的字符串，value=对应异位词分组
        unordered_map<string, vector<string>> groupMap;
        // 遍历所有字符串
        for (string& s : strs) {
            string key = s; // 拷贝原字符串用于排序，不修改原串
            sort(key.begin(), key.end()); // 对字符排序，生成唯一key
            groupMap[key].push_back(s); // 将原串加入对应分组
        }
        // 提取哈希表中的所有分组，组装成结果
        vector<vector<string>> result;
        for (auto& pair : groupMap) {
            result.push_back(pair.second);
        }
        return result;
    }
};

// 测试代码
int main() {
    Solution solution;
    // 测试用例1
    vector<string> strs1 = {"eat", "tea", "tan", "ate", "nat", "bat"};
    vector<vector<string>> res1 = solution.groupAnagrams(strs1);
    cout << "测试用例1结果：" << endl;
    for (auto& group : res1) {
        cout << "[";
        for (int i = 0; i < group.size(); ++i) {
            cout << "\"" << group[i] << "\"";
            if (i != group.size() - 1) cout << ",";
        }
        cout << "] ";
    }
    cout << endl << endl;

    // 测试用例2
    vector<string> strs2 = {""};
    vector<vector<string>> res2 = solution.groupAnagrams(strs2);
    cout << "测试用例2结果：" << endl;
    for (auto& group : res2) {
        cout << "[";
        for (int i = 0; i < group.size(); ++i) {
            cout << "\"" << group[i] << "\"";
        }
        cout << "]" << endl;
    }
    cout << endl;

    // 测试用例3
    vector<string> strs3 = {"a"};
    vector<vector<string>> res3 = solution.groupAnagrams(strs3);
    cout << "测试用例3结果：" << endl;
    for (auto& group : res3) {
        cout << "[";
        for (int i = 0; i < group.size(); ++i) {
            cout << "\"" << group[i] << "\"";
        }
        cout << "]" << endl;
    }

    return 0;
}
```

### 代码核心细节说明

1. `sort(key.begin(), key.end())`：对字符串的字符进行原地升序排序，C++ 的 string 支持迭代器，可直接使用 sort 算法。
2. 拷贝原字符串生成 key：避免修改输入的原字符串，保证分组时存入的是原始内容。
3. 哈希表的自动初始化：当 key 不存在于哈希表中时，`groupMap[key]`会自动创建一个空的 vector<string>，因此可以直接调用 push_back。

### 复杂度分析

- 时间复杂度：`O(n * k logk)`，其中`n`是字符串的数量，`k`是字符串的最大长度。每个字符串排序的时间为`O(k logk)`，共需处理`n`个字符串。
- 空间复杂度：`O(n * k)`，需要存储所有输入的字符串到哈希表中，总字符数为`n*k`。

## 三、解法 2：字符计数法（最优时间复杂度，进阶方案）

### 核心思路

字母异位词的核心特征是**每个英文字母的出现次数完全一致**。

由于题目输入均为小写英文字母（共 26 个），我们可以用一个长度为 26 的数组，统计每个字符串中 a-z 的出现次数，再将这个计数数组转换为唯一的字符串作为哈希表的 key，以此实现分组。

该方法省去了排序的开销，时间复杂度更优。

### 详细执行流程

1. 初始化哈希表：`unordered_map<string, vector<string>>`，key 为字符计数生成的唯一标识，value 为对应异位词分组。
2. 遍历输入数组中的每一个字符串：
    
    a. 初始化长度为 26 的数组`count`，初始值全为 0，对应 a-z 的计数。
    
    b. 遍历字符串的每个字符，统计每个字母的出现次数（`count[c - 'a']++`，`c-'a'`将字符映射到 0-25 的下标）。
    
    c. 将计数数组转换为唯一的字符串 key（用分隔符拼接每个计数，避免歧义）。
    
    d. 将原字符串追加到哈希表中该 key 对应的 vector 列表末尾。
3. 遍历完成后，提取哈希表中所有分组，组装成结果返回。

### 完整 C++ 代码

```cpp
#include <iostream>
#include <vector>
#include <unordered_map>
#include <string>

using namespace std;

class Solution {
public:
    vector<vector<string>> groupAnagrams(vector<string>& strs) {
        unordered_map<string, vector<string>> groupMap;
        for (string& s : strs) {
            // 初始化计数数组，26个小写字母，初始值0
            int count[26] = {0};
            for (char c : s) {
                count[c - 'a']++; // 统计每个字母出现次数
            }
            // 将计数数组转换为唯一key，用#分隔避免歧义
            string key;
            for (int i = 0; i < 26; ++i) {
                key += to_string(count[i]) + "#";
            }
            groupMap[key].push_back(s);
        }
        // 组装结果
        vector<vector<string>> result;
        for (auto& pair : groupMap) {
            result.push_back(pair.second);
        }
        return result;
    }
};

// 测试代码和排序法完全一致
int main() {
    Solution solution;
    // 测试用例1
    vector<string> strs1 = {"eat", "tea", "tan", "ate", "nat", "bat"};
    vector<vector<string>> res1 = solution.groupAnagrams(strs1);
    cout << "测试用例1结果：" << endl;
    for (auto& group : res1) {
        cout << "[";
        for (int i = 0; i < group.size(); ++i) {
            cout << "\"" << group[i] << "\"";
            if (i != group.size() - 1) cout << ",";
        }
        cout << "] ";
    }
    cout << endl << endl;

    // 测试用例2
    vector<string> strs2 = {""};
    vector<vector<string>> res2 = solution.groupAnagrams(strs2);
    cout << "测试用例2结果：" << endl;
    for (auto& group : res2) {
        cout << "[";
        for (int i = 0; i < group.size(); ++i) {
            cout << "\"" << group[i] << "\"";
        }
        cout << "]" << endl;
    }
    cout << endl;

    // 测试用例3
    vector<string> strs3 = {"a"};
    vector<vector<string>> res3 = solution.groupAnagrams(strs3);
    cout << "测试用例3结果：" << endl;
    for (auto& group : res3) {
        cout << "[";
        for (int i = 0; i < group.size(); ++i) {
            cout << "\"" << group[i] << "\"";
        }
        cout << "]" << endl;
    }

    return 0;
}
```

### 代码核心细节说明

1. `count[c - 'a']++`：将字符`a-z`映射到数组下标`0-25`，实现 O (1) 的计数更新。
2. 用`#`拼接计数：避免出现歧义，例如计数`12`和`1`、`2`，如果直接拼接会变成`12`和`12`，用`#`分隔后为`12#`和`1#2#`，保证 key 的唯一性。

### 复杂度分析

- 时间复杂度：`O(n * k)`，其中`n`是字符串的数量，`k`是字符串的最大长度。每个字符串仅需遍历一次统计计数（O (k)），生成 key 的时间固定为 O (26)=O (1)，无排序开销。
- 空间复杂度：`O(n * k)`，和排序法一致，需要存储所有输入的字符串。
