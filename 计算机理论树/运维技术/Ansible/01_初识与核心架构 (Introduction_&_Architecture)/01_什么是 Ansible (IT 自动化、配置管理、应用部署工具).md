## 1.1 Ansible 核心定义与定位

Ansible 是一款基于 Python 开发的**开源 IT 自动化引擎**，是 DevOps 体系中基础设施即代码（Infrastructure as Code, IaC）的核心实现工具之一，核心覆盖四大自动化场景：

1. 批量服务器配置管理与系统基线收敛
2. 跨环境应用程序自动化部署与发布
3. 多节点 IT 任务的流程编排与调度
4. 公有云 / 私有云 / 混合云资源的自动化编排与管理

其核心设计哲学为**极简主义**，以人类可读的声明式语法、无代理架构、原生幂等性保障为核心优势，降低 IT 自动化的落地门槛，同时满足企业级大规模基础设施的管理需求。
