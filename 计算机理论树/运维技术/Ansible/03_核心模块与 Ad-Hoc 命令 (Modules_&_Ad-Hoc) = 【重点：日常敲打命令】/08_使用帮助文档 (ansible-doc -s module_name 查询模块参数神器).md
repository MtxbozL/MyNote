
ansible-doc 是 Ansible 官方内置的模块文档查询工具，是获取模块权威、完整用法的核心渠道，覆盖所有官方模块、插件的详细文档，无需联网，本地即可查询，是学习 Ansible 模块的核心神器。

### 3.8.1 核心命令语法

```c
ansible-doc [选项] [模块名称]
```

### 3.8.2 核心常用选项

|选项|全称|核心作用|
|:--|:--|:--|
|`-s`|`--snippet`|精简模式，输出模块的核心参数列表与简短说明，快速查询模块用法，最常用|
|无选项|-|输出模块的完整官方文档，包括详细描述、参数说明、默认值、示例、返回值|
|`-l`|`--list`|列出所有已安装的可用模块|
|`-F`|`--list_files`|列出模块名称与对应的源码文件路径|
|`-h`|`--help`|查看 ansible-doc 工具的帮助信息|

### 3.8.3 标准使用示例

```bash
# 1. 查看file模块的精简核心参数
ansible-doc -s file

# 2. 查看yum模块的完整官方文档
ansible-doc yum

# 3. 搜索所有与nginx相关的模块
ansible-doc -l | grep nginx

# 4. 查看ansible-doc工具的帮助
ansible-doc -h
```

### 3.8.4 最佳实践

1. 忘记模块参数时，优先使用`ansible-doc -s 模块名`快速查询，无需记忆所有参数；
2. 学习新模块时，先通过`ansible-doc 模块名`查看完整文档与官方示例，官方示例是最佳实践；
3. 官方文档是最权威的用法说明，优先级高于网络第三方教程，避免过时或错误的用法。
