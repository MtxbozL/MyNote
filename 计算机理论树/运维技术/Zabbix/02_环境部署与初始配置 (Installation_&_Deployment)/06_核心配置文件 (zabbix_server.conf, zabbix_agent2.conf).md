## 2.6 核心配置文件详解

Zabbix 的核心运行配置由`zabbix_server.conf`（Server 端核心配置）与`zabbix_agent2.conf`（Agent 2 端核心配置）两大文件控制，所有配置修改后必须重启对应服务方可生效，生产环境必须严格管控配置文件的访问权限。

### 2.6.1 zabbix_server.conf 核心配置

该文件是 Zabbix Server 的核心配置文件，控制服务运行、数据库连接、性能调度、日志管理等核心行为，核心参数分为必配参数与基础调度参数两类。

#### 2.6.1.1 必配参数（无有效配置则服务无法启动）

|参数名|核心作用|配置规则|
|---|---|---|
|DBHost|数据库服务地址|生产环境建议使用内网 IP，禁止使用公网地址，数据库独立部署时必须配置|
|DBName|Zabbix 数据库名称|必须与初始化创建的数据库名称一致，默认值`zabbix`|
|DBUser|数据库访问用户名|必须使用 Zabbix 专用数据库用户，禁止使用 root 等高权限用户|
|DBPassword|数据库访问密码|生产环境必须配置强密码，禁止空密码，配置文件权限设置为 600，避免密码泄露|
|DBPort|数据库服务端口|MySQL 默认 3306，PostgreSQL 默认 5432，非默认端口必须显式配置|

#### 2.6.1.2 基础运行核心参数

|参数名|核心作用|生产环境配置规则|
|---|---|---|
|ListenPort|Server 监听端口|默认 10051/TCP，非必要不修改，防火墙必须放行该端口|
|LogType|日志输出类型|生产环境设置为`file`，禁止使用控制台输出|
|LogFile|日志文件路径|必须配置可写入的绝对路径，生产环境需配置日志轮转策略|
|DebugLevel|日志调试级别|生产环境默认设置为 3，故障排查时可临时调高，禁止长期开启高调试级别避免磁盘耗尽|
|Timeout|采集超时时间|默认 3s，最大不超过 30s，跨网络场景可适当调整，避免超时导致数据采集失败|
|HousekeepingFrequency|管家进程执行频率|控制历史数据清理周期，大规模环境建议降低执行频率，后续性能优化章节会详细说明|

### 2.6.2 zabbix_agent2.conf 核心配置

该文件是 Zabbix Agent 2 的核心配置文件，控制 Agent 与 Server 的通信模式、采集行为、插件管理等核心功能，核心参数与主动 / 被动模式直接关联，与第一章的工作模式体系完全对应。

#### 2.6.2.1 核心通信参数

|参数名|核心作用|配置规则|
|---|---|---|
|Server|被动模式允许的 Server/Proxy 地址|配置 Server/Proxy 的 IP 地址段，多个地址用逗号分隔，被动模式下仅接受该参数内地址的采集请求，是被动模式的必配参数|
|ServerActive|主动模式的 Server/Proxy 地址|配置 Server/Proxy 的 IP: 端口，默认端口 10051，多个地址用逗号分隔，是主动模式的必配参数，Agent 会主动向该地址上报数据|
|Hostname|主机名标识|主动模式下必须与 Zabbix Web 前端配置的主机名完全一致，用于 Server 识别监控主机，是主动模式的必配参数|
|ListenPort|Agent 监听端口|默认 10050/TCP，被动模式必须放行该端口，纯主动模式可关闭监听端口|

#### 2.6.2.2 基础运行与扩展参数

| 参数名          | 核心作用   | 配置规则                                                         |
| ------------ | ------ | ------------------------------------------------------------ |
| LogType      | 日志输出类型 | 生产环境设置为`file`，禁止控制台输出                                        |
| LogFile      | 日志文件路径 | 配置绝对路径，生产环境配置日志轮转策略                                          |
| DebugLevel   | 日志调试级别 | 生产环境默认 3，故障排查时临时调整                                           |
| Timeout      | 采集超时时间 | 默认 3s，自定义脚本采集场景可适当调整，最大不超过 30s                               |
| HostMetadata | 主机元数据  | 用于主动注册场景，配置主机特征标识，实现自动注册时的模板自动匹配                             |
| Plugins.*    | 插件配置项  | Agent 2 原生插件的配置参数，用于 MySQL、Redis、Docker 等中间件的原生采集配置，无需额外脚本扩展 |
