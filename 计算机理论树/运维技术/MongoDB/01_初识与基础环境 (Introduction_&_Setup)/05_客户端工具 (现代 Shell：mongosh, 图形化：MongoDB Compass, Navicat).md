
## 1.5 MongoDB 主流客户端工具

### 1.5.1 mongosh 现代命令行 Shell

mongosh 是 MongoDB 官方推出的现代交互式命令行客户端，为旧版`mongo` Shell 的替代产品，基于 Node.js runtime 构建，完全兼容 ES6 + 语法，是 MongoDB 开发、运维、调试的核心工具。

- 核心能力：支持全量数据库操作命令、JavaScript 脚本编写与执行、自动化运维、执行计划分析、性能诊断等全场景能力；
- 增强特性：内置语法高亮、自动补全、命令历史、格式化输出，大幅提升命令行操作效率；
- 适用场景：生产环境运维、自动化脚本开发、无 GUI 的服务器环境操作、命令行快速调试。

### 1.5.2 MongoDB Compass 官方图形化客户端

MongoDB Compass 是 MongoDB 官方推出的免费开源图形化 GUI 工具，适配全平台，是入门学习、可视化管理、性能调优的核心工具。

- 核心能力：可视化数据增删改查、Schema 自动分析与可视化、索引管理与图形化执行计划分析、拖拽式聚合管道编辑与实时预览、自动生成对应操作的 Shell 命令；
- 适用场景：MongoDB 入门学习、数据可视化分析、性能调优、非专业运维人员的日常管理。

### 1.5.3 第三方商业客户端：Navicat

Navicat 是主流的商业多数据库统一管理工具，原生支持 MongoDB 与各类主流 RDBMS 的统一管理，适配多数据库栈的企业级场景。

- 核心能力：跨数据库统一管理、异构数据库数据同步与迁移、类 SQL 语法查询 MongoDB 数据、可视化操作增强、备份恢复、任务调度等企业级运维能力；
- 适用场景：多数据库栈的开发与运维团队、企业级生产环境管理、异构数据库数据迁移场景。
