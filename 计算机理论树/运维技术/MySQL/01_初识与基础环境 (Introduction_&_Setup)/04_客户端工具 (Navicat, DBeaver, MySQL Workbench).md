
## 1.4 客户端工具

### 1.4.1 原生命令行客户端

MySQL 原生命令行客户端是所有 GUI 工具的底层基础，无额外依赖，全版本兼容，是学习、排障、运维的核心工具，核心调用格式为：

```bash
mysql -u 用户名 -p -h 主机地址 -P 端口 -D 数据库名
```

核心常用命令包括：库表操作、用户权限管理、配置查询、事务控制、性能监控等，是 MySQL 学习必须掌握的基础工具。

### 1.4.2 主流 GUI 客户端

1. **MySQL Workbench**
    
    Oracle 官方出品的免费客户端，分为社区版与商业版，完全兼容 MySQL 全版本特性，核心能力包括：SQL 编辑与执行、可视化 ER 图建模（正向 / 反向工程）、数据库管理、性能监控与调优、数据迁移、备份恢复。无版权风险，功能覆盖学习、开发、运维全流程，是入门学习的首选工具。
    
1. **Navicat**
    
    商业跨平台数据库客户端，支持 MySQL、PostgreSQL、Oracle 等多品类数据库，核心特性包括：极简可视化操作、智能 SQL 编辑提示、数据 / 结构同步、批量数据处理、备份恢复、报表生成。操作门槛低，功能全面，是开发与运维的主流商业工具，需注意商业版权合规。
    
1. **DBeaver**
    
    开源免费跨平台客户端，基于 Java 开发，支持全品类关系型与非关系型数据库，核心特性包括：标准化 SQL 编辑器、自动 ER 图生成、元数据管理、全格式数据导入导出、插件化扩展。开源无版权风险，多数据源兼容性强，适合多数据库场景的学习与开发。
