### 3.1.1 if /elif/else 核心语法

Python 通过`if/elif/else`实现分支逻辑，核心依赖**代码缩进**划分代码块（PEP8 规范推荐 4 个空格缩进，禁止 Tab 与空格混合使用），无 C/Java 等语言的大括号包裹，语法结构如下：

```python
if 条件表达式1:
    # 条件1为True时执行的代码块
    语句1
elif 条件表达式2:
    # 条件1为False、条件2为True时执行的代码块
    语句2
else:
    # 所有条件均为False时执行的代码块
    语句3
```

核心规则：

1. 条件表达式的结果会被内置函数`bool()`转换为布尔值，真值判断规则与第一章布尔型章节完全一致；
2. `if`为必选关键字，`elif`支持 0 个或多个，实现多分支判断；`else`为可选关键字，最多只能有 1 个；
3. Python 无原生`switch-case`语法，官方推荐：3 个以内分支使用`if-elif-else`，超过 5 个分支的复杂场景使用**字典映射**替代，实现 O (1) 时间复杂度的分支调度，示例：

    ```python
    def func1(): return "分支1执行"
    def func2(): return "分支2执行"
    def default(): return "默认分支执行"
    
    switch_map = {"key1": func1, "key2": func2}
    result = switch_map.get("key1", default)()
    ```
    
4. 条件表达式支持短路运算：`if a and b`中，若`a`为 False，不会执行`b`的表达式，可用于安全前置判断，如`if obj is not None and obj.attr > 0`。

### 3.1.2 三元表达式（条件表达式）

三元表达式是`if-else`分支的单行简写形式，用于简单的条件赋值场景，语法结构如下：

```python
结果1 if 条件表达式 else 结果2
```

执行逻辑：条件表达式为 True 时返回`结果1`，为 False 时返回`结果2`。

核心用法与规范：

1. 基础赋值场景：

    ```python
    # 等价于 if a > b: max_num = a else: max_num = b
    max_num = a if a > b else b
    ```
    
2. 支持嵌套，但 PEP8 规范明确禁止超过 2 层嵌套，避免可读性下降；
3. 仅适用于简单单行逻辑，复杂分支必须使用完整`if-else`结构，禁止滥用三元表达式。
