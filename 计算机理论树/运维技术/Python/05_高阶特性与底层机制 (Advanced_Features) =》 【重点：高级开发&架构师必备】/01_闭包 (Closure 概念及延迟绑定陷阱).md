闭包是 Python 高阶函数的核心基础，是装饰器、函数式编程的底层实现载体，其核心本质是**函数作用域与生命周期的延伸**。

### 5.1.1 闭包的核心定义与形成条件

在 Python 中，闭包是指**内层函数引用了外层函数作用域（Enclosing）中的非全局变量，且外层函数返回了内层函数的引用**的函数结构。闭包打破了「函数执行结束后，其作用域内的变量会被销毁」的常规规则，让外层作用域的变量在函数执行结束后依然可被访问。

形成闭包必须同时满足 3 个必要条件：

1. 必须存在**嵌套函数结构**（外层函数 + 内层函数）；
2. 内层函数必须**引用外层函数作用域中的非全局、非内层局部变量**（即自由变量）；
3. 外层函数必须**返回内层函数的对象引用**（而非函数调用结果）。

示例：标准闭包实现

```python
def outer_func(base_num):
    # 外层函数的局部变量，被内层函数引用，成为自由变量
    def inner_func(increment):
        # 内层函数引用外层的base_num变量
        return base_num + increment
    # 外层函数返回内层函数对象，而非调用结果
    return inner_func

# 外层函数执行结束，返回内层函数，赋值给adder
adder = outer_func(10)
# 外层函数已执行完毕，但其作用域的base_num变量依然被保留
print(adder(5))   # 输出：15
print(adder(3))   # 输出：13
```

### 5.1.2 闭包的核心特性与底层实现

1. **作用域生命周期延伸**：外层函数执行结束后，其作用域不会被 Python 的垃圾回收机制销毁，因为内层函数依然持有对自由变量的引用，自由变量的生命周期与内层闭包函数绑定。
2. **自由变量存储**：闭包函数的`__closure__`属性会存储所有引用的自由变量，每个变量对应一个`cell`单元格对象，通过`cell_contents`属性可获取变量的值，是闭包存在的直接证明。
    
    示例：

    ```python
    print(adder.__closure__)
    # 输出：(<cell at 0x...: int object at 0x...>,)
    print(adder.__closure__[0].cell_contents)  # 输出：10
    ```
    
3. **变量修改规则**：闭包中，内层函数仅可读取自由变量的值，若要修改自由变量的绑定，必须使用`nonlocal`关键字声明，否则会被识别为内层函数的局部变量，触发`UnboundLocalError`，与第三章作用域规则完全联动。

### 5.1.3 闭包的延迟绑定陷阱【面试高频坑点】

延迟绑定是闭包最核心的易错点，其底层原理：**Python 闭包中，内层函数对自由变量的引用是延迟绑定的，仅当内层函数执行时，才会去外层作用域查找变量的当前值，而非在内层函数定义时绑定变量的瞬时值**。

最典型的反例：循环中创建闭包

```python
def create_funcs():
    funcs = []
    for i in range(3):
        # 循环中创建闭包函数，引用循环变量i
        def inner():
            return i ** 2
        funcs.append(inner)
    return funcs

funcs = create_funcs()
# 非预期结果：所有函数执行都返回4，而非0、1、4
print(funcs)  # 输出：4
print(funcs)  # 输出：4
print(funcs)  # 输出：4
```

问题根源：循环结束后，`i`的最终值为 2，当闭包函数执行时，才会查找`i`的当前值，因此所有函数都使用最终的 2 进行计算，而非循环时的瞬时值。

**解决方案**：通过默认参数在函数定义时完成变量绑定

```python
def create_funcs():
    funcs = []
    for i in range(3):
        # 通过默认参数，在函数定义时绑定i的当前值
        def inner(num=i):
            return num ** 2
        funcs.append(inner)
    return funcs

funcs = create_funcs()
# 符合预期结果
print(funcs)  # 输出：0
print(funcs)  # 输出：1
print(funcs)  # 输出：4
```
