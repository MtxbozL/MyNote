
运算符用于对一个或多个操作数（变量、字面量、表达式）执行运算操作，Python 所有运算符均有明确的优先级与结合性，括号`()`可强制改变运算优先级。

### 1.6.1 算术运算符

用于执行数值运算，核心运算符如下：

|运算符|功能描述|示例|结果|
|---|---|---|---|
|`+`|加法|`3 + 2`|5|
|`-`|减法|`3 - 2`|1|
|`*`|乘法|`3 * 2`|6|
|`/`|浮点数除法|`5 / 2`|2.5|
|`//`|地板除法（向下取整）|`5 // 2` / `-5 // 2`|2 / -3|
|`%`|取模（求余数）|`5 % 2`|1|
|`**`|幂运算|`3 ** 2`|9|

补充：`+`与`*`可用于字符串、列表等序列类型，实现拼接与重复操作，如`"ab" + "cd" = "abcd"`，`"ab" * 3 = "ababab"`。

### 1.6.2 比较运算符

用于比较两个操作数的大小 / 值关系，返回布尔值`True`/`False`，核心运算符如下：

|运算符|功能描述|
|---|---|
|`==`|等于，比较两个操作数的值是否相等|
|`!=`|不等于，比较两个操作数的值是否不相等|
|`>`|大于|
|`<`|小于|
|`>=`|大于等于|
|`<=`|小于等于|

核心特性：Python 支持链式比较，如`1 < 2 < 3`等价于`1 < 2 and 2 < 3`，返回`True`，是 Python 区别于其他编程语言的核心特性之一。

### 1.6.3 逻辑运算符

用于布尔值的逻辑运算，支持短路运算机制，核心运算符如下：

|运算符|功能描述|短路规则|
|---|---|---|
|`and`|逻辑与，全真为真，一假则假|第一个操作数为`False`时，直接返回第一个操作数，不计算后续表达式|
|`or`|逻辑或，一真为真，全假则假|第一个操作数为`True`时，直接返回第一个操作数，不计算后续表达式|
|`not`|逻辑非，真值取反|无短路特性，返回布尔值|

示例：`0 and 5`返回`0`，`5 or 0`返回`5`，`not 3>2`返回`False`。

### 1.6.4 赋值运算符

用于为变量赋值，分为基础赋值与复合赋值，核心运算符如下：

|运算符|功能描述|等价表达式|
|---|---|---|
|`=`|基础赋值|-|
|`+=`|加法赋值|`a += 2` → `a = a + 2`|
|`-=`|减法赋值|`a -= 2` → `a = a - 2`|
|`*=`|乘法赋值|`a *= 2` → `a = a * 2`|
|`/=`|除法赋值|`a /= 2` → `a = a / 2`|
|`//=`|地板除法赋值|`a //= 2` → `a = a // 2`|
|`%=`|取模赋值|`a %= 2` → `a = a % 2`|
|`**=`|幂运算赋值|`a **= 2` → `a = a ** 2`|

补充：Python 支持多变量同时赋值，本质为元组解包，可实现无临时变量的变量交换：`a, b = b, a`。

### 1.6.5 成员运算符

用于判断一个元素是否存在于序列 / 集合 / 字典等可迭代对象中，返回布尔值，核心运算符：

- `in`：元素存在于可迭代对象中返回`True`，否则返回`False`，示例：`'a' in 'abc' → True`，`3 in [1,2,3] → True`；
- `not in`：元素不存在于可迭代对象中返回`True`，否则返回`False`，示例：`4 not in [1,2,3] → True`。

注意：对于字典，`in`默认判断键（key）是否存在，而非值（value），如`1 in {'a':1, 'b':2} → False`。

### 1.6.6 身份运算符与核心区别

身份运算符用于判断两个变量是否引用**同一个对象**，即两个变量的内存地址（`id`）是否相同，核心运算符：

- `is`：两个变量引用同一个对象返回`True`，否则返回`False`；
- `is not`：两个变量引用不同对象返回`True`，否则返回`False`。

【核心重点】`is` 与 `==` 的本质区别：

1. `==` 是**值比较运算符**：用于判断两个对象的 ** 值（内容）** 是否相等，会调用对象的`__eq__()`方法执行比较逻辑；
2. `is` 是**身份比较运算符**：用于判断两个对象的 ** 内存地址（id）** 是否相同，即是否为同一个对象，本质是判断`id(a) == id(b)`。

示例：

```python
a = [1,2,3]
b = [1,2,3]
c = a
print(a == b)  # True：两个对象的值完全相同
print(a is b)  # False：两个对象位于不同内存地址，是不同的对象
print(a is c)  # True：c与a引用同一个对象，内存地址相同
```

补充说明：

- Python 为`[-5, 256]`范围内的整数创建了常驻内存的对象池，该范围内的相同整数会复用同一个对象，因此`a=100; b=100; a is b`返回`True`；超出该范围的整数会创建新对象，不可用`is`判断相等；
- 官方推荐用法：`is`仅用于判断变量是否为`None`、`True`、`False`，如`if x is None:`，其余所有值比较场景均使用`==`。
