### 1.5.1 字符串（`str`）核心定义

Python 中字符串是由**Unicode 字符组成的不可变有序序列**，用于表示文本信息，所有字符串均为`str`类的实例。

1. 字面量表示：
    
    - 单行字符串：使用单引号`' '`或双引号`" "`包裹，两种写法无本质区别，可互相嵌套避免转义，如`"I'm a student"`无需对单引号转义；
    - 多行字符串：使用三单引号`''' '''`或三双引号`""" """`包裹，可保留换行、缩进等格式，常用于多行文本、函数 / 类的文档字符串（docstring）；
    
2. 不可变特性：字符串创建后，其内部的字符序列不可修改，对字符串的任何修改操作（拼接、替换、切片等）都会生成新的字符串对象，而非修改原对象；
3. 序列特性：字符串支持有序序列的通用操作，包括索引访问（`s[0]`获取第一个字符，`s[-1]`获取最后一个字符）、切片、长度获取`len(s)`、成员运算等。

### 1.5.2 转义字符

转义字符以反斜杠`\`开头，用于表示无法直接输入的特殊字符，或避免与字符串边界符冲突，Python 核心转义字符如下：

|转义字符|功能描述|
|---|---|
|`\n`|换行符，光标移动到下一行开头|
|`\t`|水平制表符（Tab），默认对应 4 个空格|
|`\r`|回车符，光标移动到当前行开头|
|`\\`|反斜杠本身，用于表示`\`字符|
|`\'`|单引号，用于单引号包裹的字符串中输入单引号|
|`\"`|双引号，用于双引号包裹的字符串中输入双引号|
|`\b`|退格符，删除前一个字符|

**原始字符串（Raw String）**：在字符串前加`r`/`R`，可关闭转义机制，所有字符均按字面量处理，常用于表示文件路径、正则表达式，如`r"C:\Users\Python\test.txt"`无需对`\`进行转义。

### 1.5.3 字符串格式化输出

用于将变量、表达式的值嵌入字符串中，实现动态文本生成，Python 提供三种主流格式化方案，如下：

#### 1.5.3.1 % 格式化（C 语言风格）

Python 最早支持的格式化方式，通过`%`占位符匹配对应类型的变量，语法为`"格式化字符串" % (变量元组)`。

- 核心占位符：`%d`（十进制整数）、`%f`（浮点数，`%.2f`表示保留 2 位小数）、`%s`（字符串，可匹配任意类型对象）、`%x`（十六进制整数）、`%e`（科学计数法浮点数）；
- 示例：

    ```python
    name = "Python"
    version = 3.12
    print("Hello %s, version %.2f" % (name, version))
    # 输出：Hello Python, version 3.12
    ```

- 局限性：不支持位置复用、可读性差、复杂场景易出错，新项目不推荐使用。

#### 1.5.3.2 str.format () 格式化

Python 2.6/3.0 引入的格式化方案，通过`{}`作为占位符，解决了 % 格式化的核心缺陷，兼容性极强。

- 核心用法：

    ```python
    # 1. 按位置匹配
    print("Hello {}, version {}".format(name, version))
    # 2. 按索引匹配，支持位置复用
    print("Hello {0}, version {1}, {0} is great".format(name, version))
    # 3. 按关键字匹配
    print("Hello {name}, version {ver}".format(name=name, ver=version))
    # 4. 精度控制
    print("{:.2f}".format(3.1415926))  # 输出3.14
    ```
    
- 优势：可读性强、支持位置复用、支持复杂数据结构的成员访问，跨版本兼容性好。

#### 1.5.3.3 f-string 格式化（格式化字符串字面值）

Python 3.6 引入的现代格式化方案，是**官方推荐的首选格式化方式**，语法为`f"字符串{表达式}"`，在字符串前加`f`/`F`，`{}`中可直接写入变量、表达式、函数调用，运行时动态解析并嵌入结果。

- 核心用法：

    ```python
    # 1. 基础变量嵌入与精度控制
    print(f"Hello {name}, version {version:.2f}")
    # 2. 表达式计算
    print(f"1 + 2 = {1+2}")
    # 3. 函数调用
    print(f"name upper: {name.upper()}")
    # 4. 调试用法（Python 3.8+），自动打印变量名与值
    print(f"{version=}")  # 输出：version=3.12
    ```
    
- 优势：语法简洁、可读性最高、执行效率优于前两种方案、功能全面，是目前工业界主流方案。
