Python 是由 Guido van Rossum 于 1991 年正式发布的**高级、通用、解释型、强类型动态编程语言**，核心设计哲学遵循 PEP 20（Python 之禅），以简洁、可读、可维护为核心设计准则，是目前工业界与学术界应用最广泛的编程语言之一。

### 1.1.1 解释型语言特性

Python 的执行流程为：**源代码 (.py) → 解释器编译为字节码 (.pyc) → Python 虚拟机 (PVM) 逐行执行字节码**，核心特性如下：

1. 无需预先编译为机器码，运行时由解释器动态解析执行，区别于 C/C++ 等编译型语言；
2. 跨平台兼容性强，同一套源代码可在安装了对应平台 Python 解释器的 Windows/macOS/Linux 系统上运行，实现 “一次编写，到处运行”；
3. 开发与调试效率极高，无需编译链接环节，可直接执行代码片段；
4. 执行效率低于原生编译型语言，核心性能瓶颈可通过 C 扩展、JIT 编译器等方案优化。

### 1.1.2 强类型动态类型系统

1. **动态类型核心定义**：变量的类型在运行时确定，而非编译期；变量本身不绑定固定数据类型，仅作为内存中对象的引用，可在运行时指向不同类型的对象。
    
    示例：`a = 1`（指向整型对象）→ `a = "Python"`（重新指向字符串对象），该操作在 Python 中完全合法。
2. **强类型特性**：Python 不会隐式执行不安全的类型转换，不同类型的对象执行非法运算时会直接抛出`TypeError`，区别于 JavaScript 等弱类型语言。
    
    示例：`1 + "1"`会直接触发类型错误，而非隐式转换为字符串或数值。
