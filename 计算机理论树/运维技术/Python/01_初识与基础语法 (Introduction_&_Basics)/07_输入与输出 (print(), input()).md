### 1.7.1 输出函数 print ()

Python 内置的标准输出函数，用于将指定内容输出到控制台（标准输出流`stdout`），是程序与用户交互的核心基础函数，完整语法如下：

```python
print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)
```

参数详解：

1. `*objects`：可变位置参数，可接收任意数量、任意类型的对象，多个对象用逗号分隔，print 会依次输出所有对象的字符串形式；
2. `sep`：分隔符参数，可选，默认值为空格`' '`，用于指定多个输出对象之间的分隔符；
3. `end`：结束符参数，可选，默认值为换行符`'\n'`，用于指定输出结束时追加的字符，可通过修改该参数实现不换行输出；
4. `file`：输出流参数，可选，默认值为控制台`sys.stdout`，可指定输出到文件对象，实现内容写入文件；
5. `flush`：缓冲区刷新参数，可选，默认值为`False`，设为`True`时会强制刷新输出缓冲区，立即将内容输出到目标流，常用于实时日志、进度条打印场景。

示例：

```python
# 基础多对象输出，自定义分隔符
print("Hello", "Python", 3.12, sep=" | ")
# 输出：Hello | Python | 3.12

# 不换行输出
print("Hello", end=" ")
print("World")
# 输出：Hello World

# 输出到文件
with open("output.txt", "w", encoding="utf-8") as f:
    print("Hello Python", file=f)
```

### 1.7.2 输入函数 input ()

Python 内置的标准输入函数，用于从控制台（标准输入流`stdin`）接收用户输入，程序会阻塞等待，直到用户输入完成并按下回车键，**返回值永远为字符串类型`str`**，完整语法如下：

```python
input(prompt='')
```

参数详解：

`prompt`：提示信息参数，可选，用于在控制台输出指定的提示文本，引导用户输入，无参数时无提示文本。

核心注意事项：

1. 无论用户输入的是数字、符号还是文本，`input()`的返回值永远是字符串，若需要数值类型，必须手动进行类型转换；
    
    示例：

    ```python
    age = int(input("请输入您的年龄："))  # 转换为整型
    score = float(input("请输入您的分数："))  # 转换为浮点型
    ```
    
2. 用户直接按下回车键无输入时，返回值为空字符串`""`；
3. 输入内容中的所有字符（包括空格、特殊符号）均会被完整保留，无转义处理。
