
Python 是纯面向对象语言，**一切皆对象**，所有基础数据类型均为内置类的实例，每个对象包含三个核心属性：

1. 唯一标识（`id`）：对象的内存地址，可通过`id()`函数获取，全局唯一；
2. 类型（`type`）：对象所属的类，可通过`type()`函数获取，对象创建后类型不可变；
3. 值（`value`）：对象存储的具体数据。

本章所述 4 种基础数值类型均为**不可变对象**，即对象创建后，其值不可修改；对变量的重新赋值，本质是改变变量的引用指向，而非修改原对象的值。

### 1.4.1 整型（`int`）

用于表示整数，无固定位数限制，仅受运行设备内存大小约束，不存在数值溢出问题，区别于 C/C++ 等语言的固定长度整型。

1. 字面量表示：
    
    - 十进制（默认）：如`100`、`-42`；
    - 二进制：`0b`前缀，如`0b1010`（对应十进制 10）；
    - 八进制：`0o`前缀，如`0o12`（对应十进制 10）；
    - 十六进制：`0x`前缀，如`0xA`（对应十进制 10）；
    
2. 类型转换：内置函数`int()`可将符合格式的字符串、浮点数转换为整型，默认向下取整，如`int(3.9)`返回`3`，`int("-123")`返回`-123`。

### 1.4.2 浮点型（`float`）

用于表示带小数的数值，基于**IEEE 754 双精度 64 位标准**实现，占用 8 字节（64 位），其中 1 位符号位、11 位指数位、52 位尾数位，可表示的数值范围约为 ±1.7×10^308，有效数字精度约 15-17 位。

1. 核心注意点：浮点数存在固有精度误差，如`0.1 + 0.2`的计算结果为`0.30000000000000004`，而非精确的`0.3`，该特性为 IEEE 754 标准的固有缺陷，而非 Python 语言问题；高精度十进制运算需使用 Python 内置`decimal`模块；
2. 字面量表示：十进制小数（如`3.14`、`-0.001`）、科学计数法（`e/E`表示 10 的幂，如`6.02e23`、`-1.6e-19`）；
3. 类型转换：内置函数`float()`可将符合格式的字符串、整数转换为浮点型。

### 1.4.3 布尔型（`bool`）

用于表示逻辑真值，仅包含两个字面量常量：`True`（真）、`False`（假）。Python 中`bool`类是`int`类的直接子类，`True`本质对应整型`1`，`False`对应整型`0`，可直接参与数值运算，如`True + 1`返回`2`，`False * 5`返回`0`。

1. 布尔值判定规则：内置函数`bool()`可将任意对象转换为布尔值，以下对象的布尔值默认为`False`，其余所有对象均为`True`：
    
    - 空值`None`；
    - 数值 0：`0`、`0.0`、`0j`；
    - 空序列 / 空集合：`""`、`[]`、`()`、`{}`、`set()`；
    - 实现了`__bool__()`方法且返回`False`，或实现了`__len__()`方法且返回`0`的自定义对象；
    
2. 核心应用：用于条件判断、循环控制的逻辑表达式。

### 1.4.4 复数型（`complex`）

用于表示数学中的复数，格式为`a + bj`，其中`a`为实部（`real`），`b`为虚部（`imag`），`j`为虚数单位（对应数学中的`i`，Python 中使用`j`避免与变量`i`混淆），实部与虚部均为浮点型。

1. 字面量表示：如`3+4j`、`-2j`、`5.2-3.6j`，也可通过内置函数`complex(a, b)`创建，如`complex(3,4)`返回`3+4j`；
2. 内置属性与方法：`complex`对象内置`real`属性获取实部，`imag`属性获取虚部，`conjugate()`方法返回共轭复数；
3. 应用场景：主要用于科学计算、信号处理、量子计算等专业领域。
