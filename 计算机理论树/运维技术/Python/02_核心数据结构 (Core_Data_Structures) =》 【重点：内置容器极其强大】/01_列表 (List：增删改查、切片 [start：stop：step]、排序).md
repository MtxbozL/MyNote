## 本章概述

本章为 Python 开发的核心基础，聚焦 Python 四大内置容器数据结构（列表、元组、字典、集合）的底层实现、核心操作、内存模型与适用场景，同时掌握极具 Pythonic 风格的推导式语法，深刻理解深浅拷贝的内存指针原理。Python 内置数据结构均为 C 语言高度优化实现，其功能完整性与执行效率是 Python 语言的核心优势之一，本章内容是后续算法实现、业务开发、性能优化的必备底层能力，同时为面试高频考点核心覆盖区。

### 2.1.1 核心定义与底层特性

列表是 Python 中最通用的**可变、有序、支持元素重复的线性容器**，底层为 C 实现的动态数组（`PyListObject`结构体），其核心存储的是元素对象的内存指针（引用），而非元素本身，因此支持任意数据类型的元素嵌套与动态修改。

核心底层特性：

1. **动态扩容机制**：列表创建时会分配一块连续的内存用于存储元素指针，当元素数量超过当前容量时，自动触发扩容。Python3 的扩容公式为：`new_allocated = (size_t)newsize + (newsize >> 3) + (newsize < 9 ? 3 : 6)`，即扩容后容量约为原容量的 1.125 倍，平衡内存占用与扩容开销；
2. **可变性**：列表的可变性指可原地修改内存中的元素指针，无需创建新对象，修改前后列表的内存地址（`id`）保持不变，区别于字符串、元组等不可变序列；
3. **序列协议实现**：完整支持 Python 序列协议，可通过索引、切片访问元素，支持迭代、拼接、重复等通用序列操作。

### 2.1.2 核心操作与时间复杂度

列表所有操作的时间复杂度均基于底层动态数组的特性，核心操作分类如下：

#### 增操作

|操作方法|功能描述|平均时间复杂度|注意事项|
|---|---|---|---|
|`append(obj)`|尾部追加单个元素|O(1)|仅修改尾部指针，触发扩容时为 O (n)，均摊复杂度仍为 O (1)|
|`extend(iterable)`|尾部拼接可迭代对象的所有元素|O (k)，k 为新增元素个数|区别于`append`，会将可迭代对象的元素逐个拆分追加，而非作为单个元素|
|`insert(index, obj)`|在指定索引位置插入元素|O(n)|插入位置后的所有元素指针需向后移动，索引越界抛出`IndexError`|
|`+` 运算符|拼接两个列表，返回新列表|O (n+m)，n、m 为两个列表的长度|不修改原列表，生成全新的列表对象|
|`*` 运算符|重复列表元素，返回新列表|O (n*k)，k 为重复次数|浅拷贝元素指针，重复嵌套可变对象时会出现引用共享问题|

#### 删操作

|操作方法|功能描述|平均时间复杂度|注意事项|
|---|---|---|---|
|`pop()`|尾部弹出并返回元素|O(1)|无参数时默认操作尾部，是最高效的删除方式|
|`pop(index)`|弹出指定索引的元素并返回|O(n)|指定位置后的所有元素指针需向前移动|
|`remove(obj)`|删除第一个匹配的元素|O(n)|需遍历列表查找元素，元素不存在抛出`ValueError`|
|`clear()`|清空列表所有元素|O(1)|释放元素指针数组，保留列表对象本身|
|`del` 语句|删除指定索引 / 切片的元素|O(n)|支持`del list[index]`、`del list[start:stop]`，底层为内存移动操作|

#### 改操作

- 索引赋值：`list[index] = obj`，直接修改指定位置的元素指针，O (1)，索引越界抛出`IndexError`；
- 切片赋值：`list[start:stop] = iterable`，替换指定范围的元素，O (k+n)，k 为新元素个数，n 为后续元素移动个数，支持不等长替换。

#### 查操作

- 索引访问：`list[index]`，通过下标直接定位元素指针，O (1)，支持负索引（`-1`为最后一个元素）；
- `index(obj, start, end)`：查找元素第一个匹配项的索引，O (n)，元素不存在抛出`ValueError`，可指定查找范围；
- `count(obj)`：统计元素在列表中出现的次数，O (n)，需遍历整个列表；
- `in` 成员运算：`obj in list`，判断元素是否存在，O (n)，遍历列表逐一匹配。

### 2.1.3 切片操作

切片是 Python 序列的核心高级操作，用于提取序列的子片段，语法格式为：

```python
list[start:stop:step]
```

参数规则：

1. `start`：切片起始索引（包含），默认值为 0；`step`为负数时，默认值为列表最后一个元素的索引；
2. `stop`：切片结束索引（不包含），默认值为列表长度；`step`为负数时，默认值为`-len(list)-1`；
3. `step`：切片步长，默认值为 1，不能为 0；负数表示从右向左反向切片。

核心特性与用法：

- 切片本质是创建新的列表对象，复制原列表对应范围的元素指针，属于**浅拷贝**，修改切片后的新列表不会影响原列表（嵌套可变对象除外）；
- 常用高级用法：
    
    - 列表复制：`list[:]`，生成原列表的浅拷贝副本；
    - 列表反转：`list[::-1]`，反向遍历生成新的反转列表；
    - 固定步长取数：`list[::2]`，提取所有偶数索引位置的元素；
    
- 切片赋值可实现批量增删改：`list[1:3] = []`，等价于删除索引 1-2 的元素；`list[1:1] = [1,2,3]`，等价于在索引 1 位置插入 3 个元素。

### 2.1.4 排序与反转

1. **原地排序 `sort()`**
    
    列表内置方法，直接修改原列表，无返回值，底层采用 Timsort 算法（结合归并排序与插入排序的混合稳定排序算法），最坏时间复杂度 O (n log n)，最佳时间复杂度 O (n)。
    
    核心参数：
    
    - `key`：排序规则函数，接收列表元素，返回用于排序的键值，如`key=lambda x: len(x)`按元素长度排序；
    - `reverse`：布尔值，默认`False`升序，`True`降序。
        
        示例：`nums.sort(key=lambda x: abs(x), reverse=True)`，按元素绝对值降序排序。
    
2. **全局排序 `sorted()`**
    
    Python 内置函数，接收可迭代对象，返回全新的排序后列表，不修改原对象，参数与`sort()`完全一致，支持所有可迭代对象，通用性更强。
    
3. **原地反转 `reverse()`**
    
    列表内置方法，直接反转原列表的元素顺序，O (n) 时间复杂度，无返回值；如需生成反转后的新列表，使用切片`list[::-1]`。
    

### 2.1.5 适用场景

列表是 Python 中最通用的容器，适用于：需要动态增删元素、频繁尾部操作、有序存储可重复数据、需要灵活修改内容的场景，是业务开发、算法实现的首选线性容器。
