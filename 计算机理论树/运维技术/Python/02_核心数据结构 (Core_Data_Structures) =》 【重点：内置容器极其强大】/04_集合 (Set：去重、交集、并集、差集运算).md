### 2.4.1 核心定义与底层特性

集合是**无序、元素唯一、可变的容器**，底层为 C 实现的哈希表（`PySetObject`结构体），与字典底层实现一致，区别在于集合仅存储哈希键，无对应值，所有元素必须是可哈希的不可变对象。

核心底层特性：

1. **元素唯一性**：集合自动去重，插入重复元素会被直接忽略，是 Python 中最高效的去重工具；
2. **无序性**：不保留元素插入顺序，不支持索引、切片等序列操作，Python3.7 + 未对集合做插入顺序保证；
3. **可哈希要求**：元素必须是可哈希的不可变对象，可变对象无法存入集合；
4. **不可变集合**：Python 提供`frozenset`不可变集合，创建后无法修改元素，具备可哈希性，可作为字典的键、集合的元素。

### 2.4.2 核心操作与时间复杂度

#### 创建规则

- 空集合：必须使用`set()`创建，`{}`为创建空字典，这是集合的核心语法坑点；
- 字面量创建：`{1, 2, 3}`，仅适用于非空集合；
- 类型转换：`set(iterable)`，从可迭代对象创建集合，自动去重，如`set([1,2,2,3])` → `{1,2,3}`；
- 不可变集合：`frozenset(iterable)`，创建不可变集合。

#### 增操作

|操作方法|功能描述|平均时间复杂度|
|---|---|---|
|`add(obj)`|添加单个元素|O(1)|
|`update(iterable)`|批量添加可迭代对象的所有元素|O (k)，k 为新增元素个数|

#### 删操作

|操作方法|功能描述|平均时间复杂度|注意事项|
|---|---|---|---|
|`remove(obj)`|删除指定元素|O(1)|元素不存在抛出`KeyError`|
|`discard(obj)`|删除指定元素|O(1)|元素不存在不报错，工业界推荐安全删除方式|
|`pop()`|随机删除并返回一个元素|O(1)|空集合抛出`KeyError`，无序性导致无法固定删除顺序|
|`clear()`|清空集合所有元素|O(1)||

#### 查询操作

- `in` 成员运算：`obj in set`，判断元素是否存在，平均 O (1)，最坏 O (n)，效率远高于列表的 O (n)，是集合的核心优势；
- 不支持索引、切片等有序访问操作。

### 2.4.3 集合数学运算

集合原生支持数学集合论的所有核心运算，运算符与方法双实现，执行效率极高，是集合的核心功能。

| 运算类型  | 运算符  | 方法                            | 功能描述                         |
| ----- | ---- | ----------------------------- | ---------------------------- |
| 交集    | `&`  | `intersection(*others)`       | 返回所有集合共同包含的元素                |
| 并集    | `\|` | `union(*others)`              | 返回所有集合的所有元素，自动去重             |
| 差集    | `-`  | `difference(*others)`         | 返回属于当前集合，但不属于所有其他集合的元素       |
| 对称差集  | `^`  | `symmetric_difference(other)` | 返回两个集合中不共同拥有的元素，等价于「并集 - 交集」 |
| 子集判断  | `<=` | `issubset(other)`             | 判断当前集合是否是 other 的子集          |
| 真子集判断 | `<`  | -                             | 判断当前集合是否是 other 的真子集         |
| 超集判断  | `>=` | `issuperset(other)`           | 判断当前集合是否是 other 的超集          |
| 真超集判断 | `>`  | -                             | 判断当前集合是否是 other 的真超集         |

补充：所有运算均有对应的原地修改版本，方法名带`_update`后缀，如`intersection_update()`、`difference_update()`，直接修改当前集合，而非返回新集合，内存开销更小，性能更优。

### 2.4.4 适用场景

集合适用于：数据去重、高频成员关系判断、数学集合运算（交集、差集等）、数据去重与对比、过滤重复数据等场景，在数据清洗、多数据集对比中应用广泛。
