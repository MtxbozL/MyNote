## 一、开篇：MySQL 函数的作用

MySQL 函数是数据库内置的「工具集」，用于**简化数据处理逻辑**，对数据进行计算、转换、格式化等操作，是 SQL 开发中提升效率的核心能力。

本笔记整理了日常开发中**90% 场景会用到的函数**，分为「字符串函数、数值函数、日期时间函数、聚合函数、流程控制函数」五大类，基于 `user_info`（用户表）和 `order_info`（订单表）演示。

---

## 二、字符串函数：处理文本数据

用于拼接、截取、格式化字符串，是最常用的函数类型之一。

表格

|函数名|功能|语法|示例|说明|
|:--|:--|:--|:--|:--|
|**CONCAT()**|拼接多个字符串|`CONCAT(str1, str2, ...)`|`SELECT CONCAT(user_name, '的手机号是', phone) AS '用户信息' FROM user_info;`|拼接用户名和手机号，生成新字符串|
|**SUBSTRING()**|截取字符串的一部分|`SUBSTRING(str, start, length)`|`SELECT SUBSTRING(phone, 8, 4) AS '手机号后四位' FROM user_info;`|从第 8 位开始，截取 4 位（start 从 1 开始计数）|
|**LENGTH()**|获取字符串的字节长度|`LENGTH(str)`|`SELECT user_name, LENGTH(user_name) AS '字节长度' FROM user_info;`|utf8mb4 下，中文占 3 字节，英文 / 数字占 1 字节|
|**CHAR_LENGTH()**|获取字符串的字符长度|`CHAR_LENGTH(str)`|`SELECT user_name, CHAR_LENGTH(user_name) AS '字符长度' FROM user_info;`|不管中文还是英文，一个字符算 1 个|
|**TRIM()**|去除字符串首尾的空格|`TRIM(str)`|`SELECT TRIM(' 张三 ') AS '去除空格后';`|也可指定去除的字符：`TRIM('x' FROM 'xx张三xx')`|
|**UPPER()/LOWER()**|字符串转大写 / 小写|`UPPER(str)` / `LOWER(str)`|`SELECT UPPER('hello') AS '转大写', LOWER('WORLD') AS '转小写';`|仅对英文字母有效|
|**REPLACE()**|替换字符串中的指定内容|`REPLACE(str, old_str, new_str)`|`SELECT REPLACE(phone, '138', '139') AS '替换后手机号' FROM user_info;`|把手机号中的「138」替换成「139」|

## 三、数值函数：处理数值计算

用于对数值型数据进行数学计算、四舍五入等操作。

表格

|函数名|功能|语法|示例|说明|
|:--|:--|:--|:--|:--|
|**ABS()**|求绝对值|`ABS(num)`|`SELECT ABS(-10) AS '绝对值';`|返回 10|
|**ROUND()**|四舍五入|`ROUND(num, decimal_places)`|`SELECT ROUND(order_amount, 1) AS '四舍五入后金额' FROM order_info;`|保留 1 位小数，decimal_places 为 0 则保留整数|
|**CEIL()**|向上取整（进一法）|`CEIL(num)`|`SELECT CEIL(99.1) AS '向上取整';`|返回 100，不管小数部分是多少，都进一位|
|**FLOOR()**|向下取整（去尾法）|`FLOOR(num)`|`SELECT FLOOR(99.9) AS '向下取整';`|返回 99，不管小数部分是多少，都舍去|
|**MOD()**|取余数|`MOD(num1, num2)`|`SELECT MOD(10, 3) AS '余数';`|返回 1（10÷3=3 余 1）|
|**RAND()**|生成 0-1 之间的随机数|`RAND()`|`SELECT RAND() AS '随机数';`|每次执行结果不同，生成 0≤x<1 的随机数|
|**FORMAT()**|格式化数值（千分位）|`FORMAT(num, decimal_places)`|`SELECT FORMAT(1234567.89, 2) AS '格式化后';`|返回「1,234,567.89」，带千分位分隔符|

## 四、日期时间函数：处理日期与时间

用于获取、格式化、计算日期时间，是订单、日志等场景的核心函数。

表格

|函数名|功能|语法|示例|说明|
|:--|:--|:--|:--|:--|
|**NOW()**|获取当前日期时间|`NOW()`|`SELECT NOW() AS '当前日期时间';`|返回格式：`2026-03-09 15:30:00`|
|**CURDATE()**|获取当前日期|`CURDATE()`|`SELECT CURDATE() AS '当前日期';`|返回格式：`2026-03-09`|
|**CURTIME()**|获取当前时间|`CURTIME()`|`SELECT CURTIME() AS '当前时间';`|返回格式：`15:30:00`|
|**DATE_FORMAT()**|格式化日期时间|`DATE_FORMAT(date, format)`|`SELECT DATE_FORMAT(create_time, '%Y年%m月%d日 %H:%i:%s') AS '格式化时间' FROM order_info;`|常用格式：`%Y`（4 位年）、`%m`（月）、`%d`（日）、`%H`（24 小时制时）、`%i`（分）、`%s`（秒）|
|**DATEDIFF()**|计算两个日期的天数差|`DATEDIFF(date1, date2)`|`SELECT DATEDIFF(CURDATE(), create_time) AS '下单天数' FROM order_info;`|返回 date1 - date2 的天数（正数表示 date1 晚于 date2）|
|**DATE_ADD()**|日期加指定时间间隔|`DATE_ADD(date, INTERVAL expr unit)`|`SELECT DATE_ADD(CURDATE(), INTERVAL 7 DAY) AS '7天后';`|unit 可选：`DAY`（天）、`MONTH`（月）、`YEAR`（年）、`HOUR`（小时）等|
|**DATE_SUB()**|日期减指定时间间隔|`DATE_SUB(date, INTERVAL expr unit)`|`SELECT DATE_SUB(CURDATE(), INTERVAL 1 MONTH) AS '1个月前';`|用法同 DATE_ADD|
|**YEAR()/MONTH()/DAY()**|提取日期的年 / 月 / 日|`YEAR(date)` / `MONTH(date)` / `DAY(date)`|`SELECT YEAR(create_time) AS '下单年份', MONTH(create_time) AS '下单月份' FROM order_info;`|分别提取年、月、日部分|

用于对一组数据进行统计计算，常与 `GROUP BY` 配合使用（详细可参考之前的「聚合与分组」笔记）。

表格

## 六、流程控制函数：逻辑判断（最实用）

用于在 SQL 中实现「if-else」「case-when」的逻辑判断，是业务场景中最常用的函数类型。

### 6.1 IF ()：简单的二选一判断

#### 功能

如果条件成立，返回第一个值；否则返回第二个值，相当于编程语言中的 `if-else`。

#### 语法

sql

```
IF(condition, value_if_true, value_if_false)
```

#### 示例

sql

```
-- 需求：判断订单状态，显示「已完成」或「未完成」
SELECT 
  order_id, order_no, order_amount,
  IF(order_status = '已支付', '已完成', '未完成') AS '订单状态说明'
FROM order_info;
```

---

### 6.2 IFNULL ()：处理 NULL 值

#### 功能

如果第一个值不为 NULL，返回第一个值；否则返回第二个值，用于给 NULL 值设置默认值。

#### 语法

sql

```
IFNULL(value, default_value)
```

#### 示例

sql

```
-- 需求：查询用户的邮箱，如果邮箱为 NULL，显示「未填写」
SELECT 
  user_id, user_name,
  IFNULL(email, '未填写') AS '邮箱'
FROM user_info;
```

---

### 6.3 CASE WHEN：多重判断（最强大）

#### 功能

实现多重逻辑判断，相当于编程语言中的 `if-else if-else` 或 `switch-case`，有两种写法。

#### 写法 1：简单 CASE（等值判断）

sql

```
CASE expression
  WHEN value1 THEN result1
  WHEN value2 THEN result2
  ...
  ELSE default_result
END
```

#### 写法 2：搜索 CASE（条件判断，更灵活）

sql

```
CASE
  WHEN condition1 THEN result1
  WHEN condition2 THEN result2
  ...
  ELSE default_result
END
```

#### 示例 1：简单 CASE（等值判断订单状态）

sql

```
SELECT 
  order_id, order_no,
  CASE order_status
    WHEN '待支付' THEN '请尽快支付'
    WHEN '已支付' THEN '已发货'
    WHEN '已取消' THEN '订单已关闭'
    ELSE '未知状态'
  END AS '状态说明'
FROM order_info;
```

#### 示例 2：搜索 CASE（条件判断年龄分组）

sql

```
SELECT 
  user_id, user_name, age,
  CASE
    WHEN age < 18 THEN '未成年'
    WHEN age BETWEEN 18 AND 35 THEN '青年'
    WHEN age BETWEEN 36 AND 60 THEN '中年'
    ELSE '老年'
  END AS '年龄分组'
FROM user_info;
```

## 七、常见坑与避坑指南

### 1. 字符串函数的字符集问题

- `LENGTH()` 返回字节长度，`CHAR_LENGTH()` 返回字符长度，中文场景下优先用 `CHAR_LENGTH()`；
- 示例：`LENGTH('张三')` 返回 6（utf8mb4 下中文占 3 字节），`CHAR_LENGTH('张三')` 返回 2。

---

### 2. 日期函数的格式问题

- `DATE_FORMAT()` 的格式符要正确，`%Y` 是 4 位年（2026），`%y` 是 2 位年（26），不要搞混；
- 日期计算时，`DATEDIFF()` 只计算天数差，忽略时间部分。

---

### 3. NULL 值的处理

- 任何函数与 NULL 运算，结果通常都是 NULL（除了 `IFNULL()`、`COALESCE()`）；
- 示例：`SELECT CONCAT('张三', NULL)` 返回 NULL，需用 `IFNULL()` 处理：`SELECT CONCAT('张三', IFNULL(email, ''))`。

---

### 4. 聚合函数忽略 NULL 值

- 除了 `COUNT(*)`，所有聚合函数都忽略 NULL 值；
- 示例：`AVG(age)` 会自动排除 `age` 为 NULL 的行，`COUNT(age)` 只统计非 NULL 的行数。

## 八、综合实操演练

基于 `user_info` 和 `order_info` 表，完成以下需求：

1. 拼接用户的姓名和手机号，格式为「张三（13800138000）」；
2. 截取订单号的后 6 位；
3. 计算订单金额的平均值，保留 2 位小数；
4. 格式化下单时间为「2026-03-09」格式；
5. 用 CASE WHEN 判断订单金额，显示「高金额（>100）」「中金额（50-100）」「低金额（<50）」；
6. 查询用户的邮箱，如果为 NULL，显示「未提供」。

### 参考答案

sql

```
-- 1. 拼接姓名和手机号
SELECT CONCAT(user_name, '（', phone, '）') AS '用户信息' FROM user_info;

-- 2. 截取订单号后6位
SELECT order_no, SUBSTRING(order_no, -6) AS '订单号后6位' FROM order_info;

-- 3. 计算平均订单金额，保留2位小数
SELECT ROUND(AVG(order_amount), 2) AS '平均订单金额' FROM order_info;

-- 4. 格式化下单时间
SELECT DATE_FORMAT(create_time, '%Y-%m-%d') AS '下单日期' FROM order_info;

-- 5. CASE WHEN 判断订单金额
SELECT 
  order_id, order_no, order_amount,
  CASE
    WHEN order_amount > 100 THEN '高金额'
    WHEN order_amount BETWEEN 50 AND 100 THEN '中金额'
    ELSE '低金额'
  END AS '金额等级'
FROM order_info;

-- 6. 处理邮箱的 NULL 值
SELECT user_id, user_name, IFNULL(email, '未提供') AS '邮箱' FROM user_info;
```

## 九、核心要点总结

1. 字符串函数：`CONCAT()` 拼接、`SUBSTRING()` 截取、`CHAR_LENGTH()` 取字符长度、`TRIM()` 去空格；
2. 数值函数：`ROUND()` 四舍五入、`CEIL()` 向上取整、`FLOOR()` 向下取整、`MOD()` 取余；
3. 日期时间函数：`NOW()` 当前时间、`DATE_FORMAT()` 格式化、`DATEDIFF()` 天数差、`DATE_ADD()` 日期加减；
4. 聚合函数：`COUNT()`、`SUM()`、`AVG()`、`MAX()`、`MIN()`，常与 `GROUP BY` 配合；
5. 流程控制函数：`IF()` 二选一、`IFNULL()` 处理 NULL、`CASE WHEN` 多重判断（最实用）；
6. 避坑：注意字符集、日期格式、NULL 值的处理。
