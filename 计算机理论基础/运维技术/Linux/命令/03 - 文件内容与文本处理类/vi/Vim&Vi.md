> 核心说明：Vim 是 Vi 的超集增强版，兼容所有 Vi 命令，新增语法高亮、多级撤销、插件系统、正则增强等能力，是 Linux/Unix 系统默认的终端编辑器，也是程序员高效编码的核心工具。本笔记从入门到精通，覆盖全场景核心用法，兼顾实用性与进阶性。

## 一、基础入门：安装与核心概念

### 1.1 Vim 安装

表格

|系统类型|安装命令|
|---|---|
|Ubuntu/Debian|`sudo apt update && sudo apt install vim`|
|CentOS/RHEL|`sudo yum install vim`|
|macOS|`brew install vim`|
|Windows|安装 gVim 或 WSL 内安装 Vim|

### 1.2 Vim 核心模式（灵魂知识点）

Vim 是**模式化编辑器**，不同模式对应不同操作逻辑，所有模式按 `Esc` 均可回到**普通模式**，新手必须先吃透模式切换。

![[Pasted image 20260310212943.png]]

| 模式名称       | 进入方式                     | 核心作用                               | 状态提示                                                        |
| ---------- | ------------------------ | ---------------------------------- | ----------------------------------------------------------- |
| 普通模式（命令模式） | 打开 Vim 默认进入，其他模式按`Esc`返回 | 执行光标移动、删除、复制、粘贴、撤销等命令，**无法直接输入文本** | 无特殊提示                                                       |
| 插入模式       | 普通模式按 `i/a/o/I/A/O` 等    | 直接输入 / 编辑文本，和普通编辑器一致               | 左下角 `-- INSERT --`                                          |
| 可视模式       | 普通模式按 `v`/`V`/`Ctrl+v`   | 选中文本，批量执行编辑操作                      | 左下角 `-- VISUAL --`/`-- VISUAL LINE --`/`-- VISUAL BLOCK --` |
| 命令行模式      | 普通模式按 `:`/`/`/`?`        | 执行保存、退出、查找替换、插件、Shell 命令等          | 左下角出现 `:`/`/`/`?` 提示符                                       |
| 替换模式       | 普通模式按 `R`                | 输入的字符直接覆盖光标后的原有内容                  | 左下角 `-- REPLACE --`                                         |

### 1.3 入门必学：打开 / 保存 / 退出

新手第一要务：学会打开和退出 Vim，避免卡死在编辑器内。

```bash
# 终端打开文件
vim 文件名       # 打开指定文件，不存在则新建
vim +n 文件名    # 打开文件并跳转到第n行
vim + 文件名     # 打开文件并跳转到最后一行

# 命令行模式（冒号开头，回车执行）
:w               # 保存当前文件
:w 新文件名      # 另存为新文件
:wq              # 保存并退出（等价于普通模式 ZZ）
:q               # 退出（仅当文件无修改时生效）
:q!              # 强制退出不保存（等价于普通模式 ZQ）
:w!              # 强制保存（针对只读文件，需权限）
:e!              # 放弃所有修改，重新加载当前文件
```

## 二、核心操作：光标移动（效率核心）

Vim 效率的核心是**全键盘光标控制**，无需移动手到方向键，按操作范围分级，优先掌握高频命令。

### 2.1 基础字符级移动

| `h`  | 光标左移一个字符（等价于左方向键）          |

| `j`  | 光标下移一行（等价于下方向键）            |

| `fx` | 跳转到当前行**下一个**字符 x（x 为任意字符） |
| `Fx` | 跳转到当前行**上一个**字符 x          |
| `;`  | 重复上一次 `f/F` 跳转             |
| `,`  | 反向重复上一次 `f/F` 跳转           |

### 2.2 单词级移动

| 命令          | 作用                   |
| ----------- | -------------------- |

| `b`         | 跳转到上一个单词的开头          |
| `e`         | 跳转到当前 / 下一个单词的结尾     |
| `ge`        | 跳转到上一个单词的结尾          |
| `W`/`B`/`E` | 以空格为分隔符的大单词跳转，忽略标点符号 |

### 2.3 行级移动

表格

### 2.4 文档级 & 屏幕级移动

## 三、核心编辑：增删改查、复制粘贴

Vim 编辑命令遵循**动词 + 量词 + 名词**的语法逻辑，掌握后可自由组合命令，无需死记硬背。

- 核心动词：`d`(删除 / 剪切)、`y`(复制)、`c`(修改，删除后进入插入模式)
- 量词：数字（重复次数）
- 名词：光标移动命令（范围），例：`d3w` = 删除 3 个单词，`y$` = 复制到行尾

### 3.1 文本插入（进入插入模式）

表格

|命令|作用（普通模式下执行）|
|---|---|
|`i`|在当前光标**前**插入|
|`I`|在当前行**行首非空字符前**插入|
|`a`|在当前光标**后**插入|
|`A`|在当前行**行尾**插入|
|`o`|在当前行**下方**新建空行，进入插入模式|
|`O`|在当前行**上方**新建空行，进入插入模式|
|`s`|删除光标所在字符，进入插入模式|
|`S`|删除当前整行，进入插入模式|
|`C`|删除光标到行尾的内容，进入插入模式|

### 3.2 删除 / 剪切操作

| 命令       | 作用                                |
| -------- | --------------------------------- |
| `x`      | 删除光标所在字符（等价于 Delete）              |
| `X`      | 删除光标前一个字符（等价于 Backspace）          |
| `dd`     | 删除 / 剪切当前整行                       |
| `ndd`    | 删除 / 剪切当前行及下方 n-1 行（例：3dd 删除 3 行） |
| `dw`     | 删除光标到当前单词结尾                       |
| `db`     | 删除光标到当前单词开头                       |
| `d$`/`D` | 删除光标到行尾                           |
| `d^`     | 删除光标到行首非空字符                       |
| `d0`     | 删除光标到绝对行首                         |
| `dG`     | 删除当前行到文档末尾                        |
| `dgg`    | 删除当前行到文档开头                        |

### 3.3 复制、粘贴、撤销、重做

| 命令       | 作用                           |
| -------- | ---------------------------- |
| `yy`     | 复制当前整行                       |
| `nyy`    | 复制当前行及下方 n-1 行（例：5yy 复制 5 行） |
| `yw`     | 复制光标到当前单词结尾                  |
| `y$`     | 复制光标到行尾                      |
| `yG`     | 复制当前行到文档末尾                   |
| `p`      | 在光标**后**粘贴（整行内容则粘贴到当前行下方）    |
| `P`      | 在光标**前**粘贴（整行内容则粘贴到当前行上方）    |
| `u`      | 撤销上一步操作                      |
| `Ctrl+r` | 重做（恢复撤销的操作）                  |
| `.`      | **超级神器**：重复上一次编辑操作，高频必学      |

### 3.4 可视模式（文本选择与批量编辑）

可视模式是 Vim 批量编辑的核心，支持三种选择模式，选中后可直接执行删除、复制、缩进、替换等操作。

| 命令       | 模式说明                     |
| -------- | ------------------------ |
| `v`      | 字符可视模式：按单个字符选择，适合选中部分文本  |
| `V`      | 行可视模式：按整行选择，适合选中多行代码     |
| `Ctrl+v` | 块可视模式：垂直选择矩形块，**批量编辑神器** |

#### 可视模式高频操作

- 选中后按 `d` 删除、`y` 复制、`c` 修改、`~` 大小写转换
- `>` 向右缩进、`<` 向左缩进、`=` 自动格式化缩进
- 块可视模式批量加注释示例（最常用）：
    
    1. 按 `Ctrl+v` 进入块可视模式
    2. 用 `j/k` 选中需要加注释的所有行的行首
    3. 按 `I`（大写 i）进入行首插入模式，输入注释符（`#`/`//`）
    4. 按 `Esc`，等待 1 秒，所有选中行自动添加注释

## 四、核心进阶：查找与替换

查找替换是 Vim 的核心能力，支持正则表达式，可实现全文档、指定范围的精准批量修改。

### 4.1 文本查找

| 命令     | 作用                         |
| ------ | -------------------------- |
| `/关键词` | 从光标位置**向下**查找关键词，回车执行      |
| `?关键词` | 从光标位置**向上**查找关键词，回车执行      |
| `n`    | 跳转到**下一个**匹配项              |
| `N`    | 跳转到**上一个**匹配项              |
| `*`    | 向下查找光标所在的单词（全词匹配）          |
| `#`    | 向上查找光标所在的单词（全词匹配）          |
| `:noh` | 取消搜索结果高亮（全称 `:nohlsearch`） |

#### 查找优化配置（写入 vimrc）

```bash
set hlsearch      " 高亮搜索结果
set incsearch     " 输入时实时匹配
set ignorecase    " 搜索不区分大小写
set smartcase     " 智能大小写：关键词含大写则区分大小写
set wrapscan      " 搜索到末尾后从头开始
```

### 4.2 文本替换（:s 命令）

#### 核心语法

```vim
:[范围]s/查找内容/替换内容/[标志]
```

- **范围**：不写默认仅处理当前行，常用范围：
    
    - `%`：整个文档（最常用）
    - `n,m`：第 n 行到第 m 行（例：10,20 第 10 到 20 行）
    - `.,$`：当前行到文档末尾
    - `'<,'>`：可视模式选中的内容（选中后自动填充）
    
- **标志**：控制替换规则，常用标志：
    
    - `g`：整行所有匹配项都替换（必加，否则仅替换每行第一个匹配项）
    - `c`：替换前逐个确认（y 确认 /n 跳过 /a 全部 /q 退出）
    - `i`：不区分大小写，`I`：强制区分大小写
    

#### 高频替换示例（开箱即用）

```vim
# 1. 全文档把old替换为new（最常用）
:%s/old/new/g

# 2. 全文档替换，带确认
:%s/old/new/gc

# 3. 替换10-20行的old为new
:10,20s/old/new/g

# 4. 删除所有空行
:%g/^$/d

# 5. 删除行尾所有空格
:%s/\s\+$//g

# 6. 全文档行首添加#注释
:%s/^/#/g

# 7. 全文档删除行首的#注释
:%s/^#//g

# 8. 正则分组替换：把123abc替换为abc123
:%s/\(\d\+\)\([a-z]\+\)/\2\1/g
```

## 五、高级功能：寄存器、宏、多文件操作

### 5.1 寄存器（多剪贴板）

Vim 不只有一个剪贴板，而是提供多组寄存器，用于存储不同的文本内容，避免复制内容被覆盖。

#### 基本用法

`"寄存器名 + 操作命令`，例：`"ayy` 把当前行复制到 a 寄存器，`"ap` 粘贴 a 寄存器的内容。

#### 核心寄存器分类

表格

| 寄存器           | 作用                                       |
| ------------- | ---------------------------------------- |
| 无名寄存器 `""`    | 默认寄存器，所有删除、复制的内容默认存在这里，`p/P` 默认调用        |
| 数字寄存器 `"0-"9` | `"0` 存最近一次复制内容，`"1-"9` 存最近 9 次删除内容，自动轮转  |
| 命名寄存器 `"a-"z` | 26 个用户自定义寄存器，内容不会被自动覆盖，适合长期存储            |
| 系统剪贴板寄存器 `"+` | 与系统剪贴板互通，`"+yy` 复制到系统剪贴板，`"+p` 粘贴系统剪贴板内容 |
| 黑洞寄存器 `"_`    | 内容删除到这里不会影响其他寄存器，例：`"_dd` 无痕迹删除行         |
| 只读寄存器         | `"%` 当前文件名，`".` 上次插入的文本，`":` 上次执行的命令     |

### 5.2 宏录制（批量重复操作神器）

宏可以录制一系列操作，重复执行，完美处理有规律的批量编辑任务，无需写脚本。

#### 宏的完整用法

1. **录制**：普通模式按 `q + 寄存器名(a-z)`，例：`qa`，左下角出现 `recording @a` 开始录制
2. **执行操作**：完成需要重复的编辑步骤（修改、跳转、删除等）
3. **停止录制**：按 `q`，录制结束
4. **执行宏**：普通模式按 `@ + 寄存器名`，例：`@a` 执行一次宏，`@@` 重复执行上一次宏
5. **批量执行**：`n@a` 执行 n 次宏，例：`100@a` 执行 100 次

#### 实用示例：给 100 行代码末尾添加分号

1. 光标移到第一行，按 `qa` 开始录制
2. 按 `A` 跳到行尾，输入 `;`，按 `Esc` 返回普通模式
3. 按 `j` 跳到下一行
4. 按 `q` 停止录制
5. 按 `99@a`，自动给剩余 99 行添加分号

### 5.3 多文件操作：缓冲区、窗口、标签页

Vim 支持多文件同时编辑，分为三个层级，满足不同场景需求。

#### 5.3.1 缓冲区（Buffer）

打开的每个文件都会对应一个缓冲区，存放在内存中，无需显示在窗口里，是多文件编辑的基础。

表格

| 命令               | 作用                            |
| ---------------- | ----------------------------- |
| `:ls`/`:buffers` | 列出所有缓冲区，显示编号、文件名、状态           |
| `:b 编号/文件名`      | 切换到指定缓冲区，例：`:b 2`/`:b main.c` |
| `:bn`            | 切换到下一个缓冲区                     |
| `:bp`            | 切换到上一个缓冲区                     |
| `:bd`            | 关闭当前缓冲区（不关闭 Vim）              |
| `:e 文件名`         | 在当前窗口打开新文件，新建缓冲区              |

#### 5.3.2 窗口（Window）

把 Vim 屏幕分割为多个区域，每个窗口显示一个缓冲区，适合同时查看 / 编辑多个文件。

表格

| 命令               | 作用                         |
| ---------------- | -------------------------- |
| `:sp 文件名`        | 水平分割窗口，打开新文件（不写文件名则打开当前文件） |
| `:vs 文件名`        | 垂直分割窗口，打开新文件（不写文件名则打开当前文件） |
| `Ctrl+w h/j/k/l` | 在窗口之间切换（左 / 下 / 上 / 右）     |
| `Ctrl+w w`       | 循环切换窗口                     |
| `Ctrl+w q`       | 关闭当前窗口                     |
| `Ctrl+w o`       | 关闭其他所有窗口，仅保留当前窗口           |
| `Ctrl+w =`       | 所有窗口大小均等                   |
| `Ctrl+w +/-`     | 增大 / 减小窗口高度                |
| `Ctrl+w >/<`     | 增大 / 减小窗口宽度                |

#### 5.3.3 标签页（Tab）

每个标签页可包含多个窗口，适合不同项目 / 文件组的隔离，比如一个标签页写代码，一个标签页看文档。

表格

|命令|作用|
|---|---|
|`:tabnew 文件名`|新建标签页，打开指定文件|
|`gt`/`:tabn`|切换到下一个标签页|
|`gT`/`:tabp`|切换到上一个标签页|
|`ngt`|跳转到第 n 个标签页，例：`2gt` 跳转到第二个标签页|
|`:tabc`|关闭当前标签页|
|`:tabo`|关闭其他所有标签页|

## 六、实用高级技巧

### 6.1 标记跳转

给文档位置做标记，实现跨文件、跨位置快速跳转。

表格

|命令|作用|
|---|---|
|`m + 小写字母(a-z)`|在当前位置做本地标记，仅当前文档有效，例：`ma`|
|`m + 大写字母(A-Z)`|全局标记，跨文件有效，例：`mA`|
|`' + 标记名`|跳转到标记所在的行首，例：`'a`|
|`` ` + 标记名 ``|跳转到标记的精确位置（行 + 列），例：`` `a ``|
|`:marks`|查看所有标记|

### 6.2 代码折叠

长代码文件必备，把函数、类折叠起来，提升阅读效率。

表格

|命令|作用|
|---|---|
|`zc`|折叠当前代码块|
|`zo`|打开当前折叠|
|`zC`|递归折叠所有嵌套代码块|
|`zO`|递归打开所有嵌套折叠|
|`za`|切换折叠状态（打开 / 关闭）|
|`zM`|折叠整个文档所有代码块|
|`zR`|打开整个文档所有折叠|

### 6.3 缩进与格式化

表格

|命令|作用|
|---|---|
|`>>`|当前行向右缩进一个单位|
|`<<`|当前行向左缩进一个单位|
|`n>>`/`n<<`|对 n 行执行缩进 / 反缩进|
|`=`|自动格式化选中内容的缩进|
|`gg=G`|自动格式化整个文档（写代码高频必用）|

### 6.4 Shell 命令集成

Vim 可直接执行 Shell 命令，无需退出编辑器。

表格

|命令|作用|
|---|---|
|`:! 命令`|执行 Shell 命令，例：`:! ls` 列出当前目录文件|
|`:r ! 命令`|把命令的输出结果插入到光标位置，例：`:r ! date` 插入当前时间|
|`:w !sudo tee %`|**高频必备**：只读文件强制保存，解决修改系统文件无权限的问题|

### 6.5 大小写转换

表格

|命令|作用|
|---|---|
|`~`|转换光标所在字符的大小写，光标右移|
|可视模式选中 + `~`|转换选中内容的大小写|
|`gu + 范围`|把指定范围内容转小写，例：`guaw` 单词转小写，`guu` 整行转小写|
|`gU + 范围`|把指定范围内容转大写，例：`gUaw` 单词转大写，`gUU` 整行转大写|

Vim 的强大之处在于插件生态，可扩展为全功能 IDE，当前主流插件管理器为 **vim-plug**，轻量高效。

#### Linux/macOS

```

#### Windows PowerShell

iwr -useb https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim |`
    ni $HOME/vimfiles/autoload/plug.vim -Force

### 8.2 插件配置与使用

1. 在 vimrc 中添加插件配置框架，把插件写在 `plug#begin` 和 `plug#end` 之间
2. 保存配置，执行 `:source ~/.vimrc` 重载配置
3. 执行 `:PlugInstall` 自动安装所有插件
4. 卸载插件：删除对应插件行，执行 `:PlugClean`
5. 更新插件：执行 `:PlugUpdate`

```
" 插件配置框架

" 主题插件
Plug 'morhetz/gruvbox'

## 九、常见问题与解决方案

1. **按 Ctrl+s 后 Vim 卡死**
    
    解决：Ctrl+s 是终端流控制锁，按 `Ctrl+q` 即可解锁恢复，不要用 Ctrl+s 作为保存快捷键。
    
2. **打开文件提示 Swap file ".xxx.swp" already exists**
    
    解决：swp 是异常退出生成的临时文件，无未保存修改按 `D` 直接删除；有修改按 `R` 恢复，保存后手动删除 swp 文件。
    
3. **只读文件修改后无法保存，提示 Permission denied**
    
    解决：执行 `:w !sudo tee %`，输入 sudo 密码即可强制保存，无需重新打开文件。
    
4. **粘贴文本后格式错乱，缩进异常**
    
    解决：粘贴前执行 `:set paste` 进入粘贴模式，粘贴完成后执行 `:set nopaste` 关闭，粘贴模式会关闭自动缩进，避免格式错乱。
    
5. **系统剪贴板 `"+y`/`"+p` 无效**
    
    解决：安装支持剪贴板的 Vim 版本，Ubuntu 执行 `sudo apt install vim-gtk3`，CentOS 执行 `sudo yum install vim-X11`，执行 `vim --version | grep clipboard`，显示 `+clipboard` 即为支持。
    
6. **中文显示乱码**
    
    解决：在 vimrc 中添加编码配置，同时确保终端编码为 UTF-8。

## 十、学习路径建议

1. **入门阶段（1 天）**
    
    掌握 6 种核心模式切换、基础打开 / 保存 / 退出、基础光标移动、增删改查，能用 Vim 完成简单文本编辑。
    
2. **进阶阶段（1-2 周）**
    
    掌握 Vim 命令语法、可视模式批量编辑、查找替换、. 重复命令、多窗口操作，日常编辑效率超过普通编辑器。
    
3. **精通阶段（长期）**
    
    掌握寄存器、宏录制、高级技巧，自定义 vimrc 配置，学会使用插件，把 Vim 打造成专属的高效 IDE。
    

> 核心提示：Vim 是用出来的，不是背出来的。优先掌握 20% 的高频命令，就能完成 80% 的操作，剩余功能在日常使用中逐步学习即可。
